# Python Курс — Telegram Бот

Интерактивный Telegram бот для изучения Python. 39 уроков в 8 модулях — от арифметики до фриланса. SQLite-хранилище, тесты, сертификат, админка.

## Возможности

- **39 структурированных уроков** в 8 тематических модулях
- **Тесты** — проверка знаний после уроков (10 уроков с тестами)
- **Сертификат** — автоматически при прохождении всего курса
- **Прогресс** — визуальный прогресс-бар со статистикой по модулям
- **Streak** — отслеживание дней подряд
- **Закладки** — сохранение уроков для быстрого доступа
- **Случайный урок** — переход к случайному непройденному уроку
- **Настройки** — настройка времени напоминаний (6:00–22:00)
- **Экспорт** — выгрузка прогресса в текстовый файл
- **Админ-панель** — статистика пользователей, рассылка сообщений
- **Ежедневные напоминания** — автоматическое уведомление
- **SQLite** — надёжное хранение без потери данных

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и главное меню |
| `/stats` | Подробная статистика |
| `/random` | Случайный непройденный урок |
| `/bookmark` | Закладки |
| `/settings` | Настройки напоминаний |
| `/export` | Экспорт прогресса |
| `/reset` | Сбросить прогресс |
| `/help` | Справка по командам |
| `/admin` | Админ-панель (только для ADMIN_IDS) |
| `/broadcast <текст>` | Рассылка всем пользователям |

## Быстрый старт

```bash
# Клонировать
git clone https://github.com/Rininganx/python-course-bot.git
cd python-course-bot

# Установить зависимости
pip install -r requirements.txt

# Задать токен (обязательно)
export TELEGRAM_BOT_TOKEN="твой_токен_бота"

# Задать ID админов (опционально)
export ADMIN_IDS="123456789,987654321"

# Запустить
python bot.py
```

### Windows (PowerShell)

```powershell
$env:TELEGRAM_BOT_TOKEN="твой_токен"
$env:ADMIN_IDS="123456789"
python bot.py
```

## Бесплатный деплой на Render.com

1. Fork репозитория
2. [render.com](https://render.com) → New → Web Service
3. Подключить GitHub репозиторий
4. Добавить переменные окружения:
   - `TELEGRAM_BOT_TOKEN` — токен бота из @BotFather
   - `ADMIN_IDS` — твой Telegram ID (узнать через @userinfobot)
5. Deploy

## Кастомизация

### Свои уроки

1. Замени `.md` файлы в `lessons/`
2. Обнови `LESSONS_ORDER` в `core/config.py`
3. Обнови `MODULES` в `core/config.py`

### Свои тесты

Добавь вопросы в `QUIZZES` в `core/handlers.py`:

```python
QUIZZES = {
    0: [  # индекс урока из LESSONS_ORDER
        {
            "q": "Вопрос?",
            "options": ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
            "answer": 0,  # индекс правильного ответа
        },
    ],
}
```
