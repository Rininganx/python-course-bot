# Арифметические операторы

> Следующая тема: 02 - Операторы присваивания — те же операторы, но с `=` в одном шаге.

## Главная идея

Python умеет считать как калькулятор. Все операторы работают с двумя числами и возвращают результат. Базовые переменные для примеров:

```python
a = 6
b = 3
```

────────────────────
## Все операторы

Оператор • Пример • Результат • Что делает
`+` • `a + b` • `9` • Сложение
`-` • `a - b` • `3` • Вычитание
`*` • `a * b` • `18` • Умножение
`/` • `a / b` • `2.0` • Деление — **всегда float**
`//` • `10 // 3` • `3` • Целочисленное деление, остаток отброшен
`%` • `a % b` • `0` • Остаток от деления (modulo)
`**` • `a ** b` • `216` • Возведение в степень (6³ = 6×6×6)

────────────────────
## Примеры кода

```python
a = 6
b = 3

print(a + b)   # → 9
print(a - b)   # → 3
print(a * b)   # → 18
print(a / b)   # → 2.0  ← не 2, а 2.0 — обрати внимание
print(a ** b)  # → 216  (6 * 6 * 6)
```

```python
# Целочисленное деление и остаток
result = 10 // 3
print(result)  # → 3  (не 3.333...)

print(10 % 3)  # → 1  (10 = 3×3 + 1, остаток = 1)
print(a % b)   # → 0  (6 делится на 3 без остатка)
```

## Практика — расчёт для PC Booster

Вот как арифметика выглядит в реальном коде. Простой расчёт процента занятой памяти:

```python
total_ram = 16384   # MB (16 GB)
used_ram  = 11000   # MB

free_ram     = total_ram - used_ram          # вычитание → 5384 MB
usage_pct    = used_ram / total_ram * 100    # деление   → 67.0%
free_gb      = free_ram // 1024              # целое деление → 5 GB
leftover_mb  = free_ram % 1024              # остаток → 360 MB

print(f"Свободно: {free_gb} GB и {leftover_mb} MB")
# → Свободно: 5 GB и 360 MB
```

────────────────────
## Заметки на полях

⚠️ `/` всегда даёт float
> `6 / 3` вернёт `2.0`, а не `2`. Если нужно целое — используй `//`. Это частая ловушка.

💡 Как запомнить `//` и `%`
> Они работают вместе: `10 // 3 = 3` (сколько раз входит), `10 % 3 = 1` (что осталось).
> Проверка: `(10 // 3) * 3 + (10 % 3)` всегда равно `10`.

📝 `**` в компьютерной графике
> Степень двойки — основа всего в IT: разрешения, размеры текстур, объём памяти.
> `2 ** 10 = 1024` (1 KB), `2 ** 20 = 1048576` (1 MB).

────────────────────
## Порядок операций

Python считает в том же порядке что и математика:
1. `**` — степень (первой!)
2. `*`, `/`, `//`, `%` — умножение и деление
3. `+`, `-` — сложение и вычитание

```python
print(2 + 3 * 4)    # → 14,  а НЕ 20
print((2 + 3) * 4)  # → 20   скобки меняют порядок
print(2 ** 3 ** 2)   # → 512  (степень справа налево: 3**2=9, потом 2**9=512)
```

────────────────────
## Практика — конвертер размеров файлов

```python
# Сколько это в мегабайтах?
bytes_count = 1536000

kb = bytes_count / 1024        # → 1500.0 KB
mb = kb / 1024                  # → 1.46484375 MB
gb = mb / 1024                  # → 0.001430... GB

print(f"{bytes_count} байт = {kb:.0f} KB = {mb:.2f} MB")

# Округление
print(round(mb, 2))             # → 1.46
print(int(mb))                  # → 1 (отбрасывает дробную часть)
```

## Упражнения — попробуй сам

1. Посчитай: сколько секунд в сутках? (24 часа * 60 минут * 60 секунд)
2. У тебя 8 GB RAM. Сколько это в мегабайтах? В килобайтах?
3. Диск 512 GB, занято 340 GB. Сколько свободно в процентах?
4. Игра весит 45 GB. Интернет 100 Мбит/с. Сколько минут скачать? (1 Мбит = 0.125 МБ)

Ответы:
```python
# 1
print(24 * 60 * 60)  # → 86400

# 2
ram_mb = 8 * 1024
ram_kb = ram_mb * 1024
print(f"8 GB = {ram_mb} MB = {ram_kb} KB")

# 3
free_pct = (512 - 340) / 512 * 100
print(f"Свободно: {free_pct:.1f}%")  # → 33.6%

# 4
size_mb = 45 * 1024              # 46080 MB
speed_mb = 100 / 8               # 12.5 МБ/с
time_sec = size_mb / speed_mb
time_min = time_sec / 60
print(f"Скачается за {time_min:.0f} минут")  # → 61 минута
```
