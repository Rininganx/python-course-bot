# Арифметика

> Python умеет считать как калькулятор — все операторы работают с двумя числами и возвращают результат.

## Theory

Операторы:
- `+` — сложение
- `-` — вычитание
- `*` — умножение
- `/` — деление (всегда float!)
- `//` — целочисленное деление (остаток отброшен)
- `%` — остаток от деления (modulo)
- `**` — возведение в степень

Порядок операций:
1. `**` — степень (первой!)
2. `*`, `/`, `//`, `%` — умножение и деление
3. `+`, `-` — сложение и вычитание

Скобки `()` меняют порядок как в математике.

## Code

```python
a = 6
b = 3

print(a + b)   # → 9
print(a - b)   # → 3
print(a * b)   # → 18
print(a / b)   # → 2.0  (не 2!)
print(a ** b)  # → 216  (6*6*6)
print(10 // 3) # → 3
print(10 % 3)  # → 1

# Порядок операций
print(2 + 3 * 4)    # → 14,  а НЕ 20
print((2 + 3) * 4)  # → 20   скобки меняют порядок
print(2 ** 3 ** 2)   # → 512  степень справа налево

# Реальный пример — расчёт памяти
total_ram = 16384
used_ram = 11000
free_gb = (total_ram - used_ram) // 1024
leftover_mb = (total_ram - used_ram) % 1024
print(f"Свободно: {free_gb} GB и {leftover_mb} MB")
```

## Practice

1. Сколько секунд в сутках? (24 * 60 * 60)
2. У тебя 8 GB RAM. Сколько это в мегабайтах?
3. Диск 512 GB, занято 340 GB. Сколько свободно в процентах?
4. Игра весит 45 GB. Интернет 100 Мбит/с. Сколько минут скачать? (1 Мбит = 0.125 МБ)

## Answers

```python
# 1
print(24 * 60 * 60)  # → 86400

# 2
ram_mb = 8 * 1024
print(f"8 GB = {ram_mb} MB")  # → 8192

# 3
free_pct = (512 - 340) / 512 * 100
print(f"Свободно: {free_pct:.1f}%")  # → 33.6%

# 4
size_mb = 45 * 1024
speed_mb = 100 / 8
time_min = size_mb / speed_mb / 60
print(f"Скачается за {time_min:.0f} минут")  # → 61
```

## Tips

- `/` всегда даёт float: `6 / 3` вернёт `2.0`, а не `2`. Если нужно целое — используй `//`
- `//` и `%` работают вместе: `10 // 3 = 3` (сколько раз), `10 % 3 = 1` (что осталось). Проверка: `(10 // 3) * 3 + (10 % 3)` всегда равно `10`
- Степень двойки — основа всего в IT: `2 ** 10 = 1024` (1 KB), `2 ** 20 = 1048576` (1 MB)
- Степень в Python считается справа налево: `2 ** 3 ** 2` = `2 ** 9` = 512, а не `8 ** 2` = 64
