# Операторы присваивания

> Связано с: 01 - Арифметика — операторы присваивания это арифметика + `=` в одном шаге.

## Главная идея

Любой оператор присваивания — это сокращение. Формула простая:

```
x op= y   →   x = x op y
```

Вместо того чтобы писать переменную дважды, пишешь один раз. Меньше кода — меньше опечаток.

────────────────────
## Все операторы

Оператор • Пример • Что делает • Развёрнутая форма
`+=` • `x += 5` • Прибавляет 5 к x • `x = x + 5`
`-=` • `x -= 5` • Вычитает 5 из x • `x = x - 5`
`*=` • `x *= 3` • Умножает x на 3 • `x = x * 3`
`/=` • `x /= 3` • Делит x на 3 (результат — float) • `x = x / 3`
`//=` • `x //= 3` • Целочисленное деление, без остатка • `x = x // 3`
`%=` • `x %= 3` • Остаток от деления • `x = x % 3`
`**=` • `x **= 2` • Возводит x в степень • `x = x ** 2`

> `+=` и `//=` и `**=` — не было в материале, добавил сам. Они работают по той же логике.

────────────────────
## Примеры кода

```python
x = 10

x -= 5   # x теперь 5  (10 - 5)
x *= 3   # x теперь 15 (5 * 3)
x /= 3   # x теперь 5.0 (15 / 3) — обрати внимание: float!
x %= 3   # x теперь 2.0 (5.0 % 3 = 2.0)

print(x)  # 2.0
```

## Практика — счётчик для PC Booster

Вот как это выглядит в реальном коде. Представь простой оптимизатор:

```python
freed_mb = 0

freed_mb += 256   # очистили кэш браузера
freed_mb += 128   # удалили temp файлы
freed_mb += 512   # очистили папку загрузок

print(f"Освобождено: {freed_mb} MB")  # → Освобождено: 896 MB
```

`+=` здесь — самый частый оператор. Каждый шаг добавляет к накопителю. Это паттерн который ты будешь использовать постоянно: счётчики, накопители, суммы.

────────────────────
## Заметки на полях

**Почему не просто `x = x - 5`?**
Технически одно и то же. Но `x -= 5` безопаснее — сложнее случайно написать `y = x - 5` и потерять изменение. Когда переменная называется длинно — `total_freed_memory_mb` — сокращение особенно спасает.

**`/=` всегда даёт float**
Даже если делится нацело: `x = 9; x /= 3` → `x = 3.0`, не `3`. Если нужно целое — используй `//=`.

────────────────────
## Строки и `+=`

Оператор `+=` работает и со строками — склеивает текст:

```python
log = ""
log += "Запуск системы...\n"
log += "Загрузка модулей...\n"
log += "Готово!\n"

print(log)
# Запуск системы...
# Загрузка модулей...
# Готово!
```

Это называется **конкатенация** (склейка). Работает только между строками:
```python
x = "Число: "
x += 42  # ❌ TypeError! Нельзя склеить строку и число
x += str(42)  # ✅ Теперь оба str
```

## Практика — бюджет ПК

```python
budget = 0

budget += 15000   # Процессор
budget += 25000   # Видеокарта
budget += 8000    # Оперативка
budget += 5000    # SSD
budget += 4000    # Блок питания
budget += 3000    # Корпус

print(f"Итого: {budget} руб")       # → Итого: 60000 руб

# Скидка 10%
budget *= 0.9
print(f"Со скидкой: {budget:.0f} руб")  # → Со скидкой: 54000 руб
```

## Упражнения — попробуй сам

1. Счётчик задач: начни с 0, прибавь 5, потом 3, потом 12. Сколько всего?
2. Начни с 1000 MB. Вычти 256, потом 128, потом 512. Сколько осталось?
3. Строка-отчёт: начни с пустой строки, добавь 3 строки через `+=`, потом выведи.
4. Температура CPU: начни с 45.0, прибавь 12.5, потом 8.3, потом вычти 15.0. Сколько?

Ответы:
```python
# 1
tasks = 0
tasks += 5
tasks += 3
tasks += 12
print(f"Задач: {tasks}")  # → 20

# 2
ram = 1000
ram -= 256
ram -= 128
ram -= 512
print(f"Осталось: {ram} MB")  # → 104

# 3
report = ""
report += "CPU: OK\n"
report += "RAM: OK\n"
report += "Disk: OK\n"
print(report)

# 4
temp = 45.0
temp += 12.5
temp += 8.3
temp -= 15.0
print(f"Температура: {temp}°C")  # → 50.8°C
```
