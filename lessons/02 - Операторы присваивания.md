# Операторы присваивания

> Сокращение: вместо `x = x + 5` пиши `x += 5`. Меньше кода — меньше опечаток.

## Theory

Формула: `x op= y` эквивалентно `x = x op y`

Операторы:
- `+=` — прибавить
- `-=` — вычесть
- `*=` — умножить
- `/=` — разделить (результат float)
- `//=` — целочисленное деление
- `%=` — остаток от деления
- `**=` — возведение в степень

Почему не `x = x - 5`? Технически одно и то же, но `x -= 5` безопаснее — сложнее случайно потерять изменение.

## Code

```python
x = 10
x -= 5   # x = 5  (10 - 5)
x *= 3   # x = 15 (5 * 3)
x /= 3   # x = 5.0 (15 / 3) — float!
x %= 3   # x = 2.0 (5.0 % 3 = 2.0)
print(x)  # 2.0

# Счётчик освобождённой памяти
freed_mb = 0
freed_mb += 256   # кэш браузера
freed_mb += 128   # temp файлы
freed_mb += 512   # папка загрузок
print(f"Освобождено: {freed_mb} MB")  # 896 MB

# += со строками (конкатенация)
log = ""
log += "Запуск системы...\n"
log += "Готово!\n"
print(log)
```

## Practice

1. Счётчик задач: начни с 0, прибавь 5, потом 3, потом 12. Сколько всего?
2. Начни с 1000 MB. Вычти 256, потом 128, потом 512. Сколько осталось?
3. Строка-отчёт: начни с пустой строки, добавь 3 строки через `+=`, выведи.
4. Температура CPU: начни с 45.0, прибавь 12.5, потом 8.3, вычти 15.0.

## Answers

```python
# 1
tasks = 0
tasks += 5
tasks += 3
tasks += 12
print(f"Задач: {tasks}")  # → 20

# 2
ram = 1000
ram -= 256
ram -= 128
ram -= 512
print(f"Осталось: {ram} MB")  # → 104

# 3
report = ""
report += "CPU: OK\n"
report += "RAM: OK\n"
report += "Disk: OK\n"
print(report)

# 4
temp = 45.0
temp += 12.5
temp += 8.3
temp -= 15.0
print(f"Температура: {temp}°C")  # → 50.8
```

## Tips

- `/=` всегда даёт float: `x = 9; x /= 3` → `3.0`, не `3`. Если нужно целое — используй `//=`
- `+=` со строками — конкатенация (склейка). Но нельзя склеить строку и число: `x = "Число: "; x += 42` → TypeError. Нужно `x += str(42)`
- `+=` — самый частый оператор в Python. Каждый шаг добавляет к накопителю: счётчики, суммы, логи
