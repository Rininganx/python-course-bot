## Главная идея

Переменная — это коробка с именем. Кладёшь данные внутрь, потом обращаешься по имени. Python сам определяет тип — не нужно объявлять заранее.

────────────────────
## Четыре основных типа

Тип • Пример • Что хранит
`int` • `ram_gb = 16` • Целое число
`float` • `cpu_temp = 72.5` • Дробное число
`str` • `pc_name = "MyPc"` • Текст (в кавычках)
`bool` • `boosted = False` • Только `True` или `False`

```python
ram_gb  = 16            # int
cpu_temp = 72.5         # float
pc_name  = "MyPc"       # str
boosted  = False        # bool

print(type(ram_gb))     # → <class 'int'>
print(type(cpu_temp))   # → <class 'float'>
print(type(pc_name))    # → <class 'str'>
print(type(boosted))    # → <class 'bool'>
```

────────────────────
## F-string — вставка переменных в текст

Пиши `f` перед кавычками, переменные вставляй через `{}`.

```python
name = "MyPc"
ram  = 16

print(f"Компьютер: {name}")       # → Компьютер: MyPc
print(f"RAM: {ram} GB")           # → RAM: 16 GB
print(f"[{name}] RAM: {ram} GB")  # → [MyPc] RAM: 16 GB
```

> Без `f` — просто текст, фигурные скобки не работают:
> `print("{name}")` выведет буквально `{name}`, а не значение.

────────────────────
## Решение задачи урока

```python
ram_gb   = 16
cpu_temp = 72.5
pc_name  = "MyPc"
boosted  = False

print(f"[{pc_name}] RAM: {ram_gb} GB")
print(f"Temp: {cpu_temp}°C")
print(f"Boost: {boosted}")
```

Вывод:
```
[MyPc] RAM: 16 GB
Temp: 72.5°C
Boost: False
```

────────────────────
## Мои ошибки на этом уроке

🐛 Ошибка 1 — переменная внутри f-string без имени
> Написал `f"{MyPC}"` вместо `f"{pc_name}"`.
> Внутри `{}` всегда пишется **имя переменной**, а не произвольный текст.
> `MyPC` — это не переменная, Python её не знает → `NameError`.

🐛 Ошибка 2 — пропущена закрывающая скобка
> Написал `print(f"[{pc_name}] RAM: {ram_gb} GB"` — нет `)` в конце.
> Каждый `print(` должен закрываться `)`. Это `SyntaxError`.
> Лайфхак: пиши сначала `print()`, потом вставляй содержимое внутрь.

────────────────────
## Заметки на полях

💡 Как проверить тип переменной
> `type(переменная)` — покажет тип. Полезно когда не уверен что внутри.

⚠️ `False` и `"False"` — разные вещи
> `False` (без кавычек) — это `bool`.
> `"False"` (в кавычках) — это `str`, просто текст.
> Python их не перепутает, но ты можешь.

📝 Связь с PC Booster
> В реальном приложении переменные типов `int/float/bool` будут везде:
> `ram_total`, `cpu_load`, `is_optimized`. F-string соберёт из них читаемый отчёт для пользователя.

## Преобразование типов

Python не смешивает типы сам. Нужно конвертировать вручную:

```python
# str → int
age_str = "25"
age_int = int(age_str)     # теперь можно считать
print(age_int + 5)         # → 30

# int → str
ram = 16
print("RAM: " + str(ram) + " GB")  # → RAM: 16 GB

# str → float
temp_str = "72.5"
temp = float(temp_str)
print(temp - 10)           # → 62.5

# int → float → int
x = float(10)      # → 10.0
y = int(7.9)       # → 7 (НЕ округляет, отбрасывает дробную часть!)
z = round(7.9)     # → 8 (вот это округляет)
```

⚠️ Что нельзя конвертировать:
```python
int("hello")    # ❌ ValueError — "hello" не число
int("12.5")     # ❌ ValueError — сначала float, потом int
float("12.5")   # ✅ → 12.5
```

## Правила именования

```python
# ✅ Хорошо
cpu_temp = 72.5
ram_total_gb = 16
is_running = True
user_name = "Alex"

# ❌ Плохо
cpuTemp = 72.5        # camelCase — не по PEP8
ram total = 16        # пробел в имени — SyntaxError
2name = "Alex"        # начинается с цифры — SyntaxError
class = "Python"      # зарезервированное слово — SyntaxError

# 💡 Совет: именуй так, чтобы было понятно без комментариев
free_disk_space_gb = 500  # ✅ Сразу понятно что и в чём измеряется
x = 500                    # ❌ Что за 500? Мегабайты? Рубли?
```

## Практика — профиль системы

```python
# Собираем информацию о системе в переменные
pc_name    = "Gaming-PC"
cpu_cores  = 8
cpu_temp   = 65.3
ram_gb     = 32
ram_used   = 18.7
gpu_name   = "RTX 4070"
gpu_temp   = 58.0
is_boosted = True

# Собираем отчёт через f-string
report = f"""
═══ Профиль системы ═══
ПК:        {pc_name}
CPU:       {cpu_cores} ядер, {cpu_temp}°C
RAM:       {ram_used}/{ram_gb} GB ({ram_used/ram_gb*100:.0f}%)
GPU:       {gpu_name}, {gpu_temp}°C
Буст:      {is_boosted}
════════════════════════
"""
print(report)
```
