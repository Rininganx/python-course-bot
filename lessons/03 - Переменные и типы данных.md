# Переменные и типы данных

> Переменная — это коробка с именем. Кладёшь данные внутрь, потом обращаешься по имени. Python сам определяет тип.

## Theory

Четыре основных типа:
- `int` — целое число: `ram_gb = 16`
- `float` — дробное число: `cpu_temp = 72.5`
- `str` — текст в кавычках: `pc_name = "MyPc"`
- `bool` — только `True` или `False`: `boosted = False`

Проверка типа: `type(переменная)` покажет тип.

F-строка: пиши `f` перед кавычками, переменные через `{}`: `f"RAM: {ram} GB"`

Преобразование типов:
- `int("25")` → 25 (str → int)
- `str(16)` → "16" (int → str)
- `float("72.5")` → 72.5 (str → float)
- `int(7.9)` → 7 (отбрасывает дробь, НЕ округляет!)
- `round(7.9)` → 8 (вот это округляет)

Именование: `snake_case` (cpu_temp), без пробелов, без цифр в начале, без зарезервированных слов.

## Code

```python
ram_gb = 16
cpu_temp = 72.5
pc_name = "MyPc"
boosted = False

print(type(ram_gb))     # <class 'int'>
print(f"[{pc_name}] RAM: {ram_gb} GB")  # [MyPc] RAM: 16 GB

# Преобразование типов
age_str = "25"
age_int = int(age_str)
print(age_int + 5)  # 30

# Округление
x = int(7.9)    # 7 (отбрасывает дробь)
y = round(7.9)  # 8 (округляет)

# Нельзя конвертировать:
# int("hello")  → ValueError!
# int("12.5")   → ValueError! (сначала float)
```

## Practice

1. Создай переменные: имя, возраст, город. Выведи одной строкой через f-string.
2. Преобразуй строку `"1024"` в число и посчитай KB (дели на 1024).
3. Создай `is_admin = True`. Выведи: `"Админ: Да"` если True.
4. Попробуй `int("3.14")` — что произойдёт? Как исправить?

## Answers

```python
# 1
name = "Alex"
age = 25
city = "Berlin"
print(f"{name}, {age} лет, {city}")

# 2
mb_str = "1024"
mb = int(mb_str)
gb = mb / 1024
print(f"{mb} MB = {gb} GB")  # 1024 MB = 1.0 GB

# 3
is_admin = True
print(f"Админ: {'Да' if is_admin else 'Нет'}")

# 4
# int("3.14") → ValueError!
# Решение: сначала float, потом int
result = int(float("3.14"))
print(result)  # 3
```

## Tips

- `False` (без кавычек) — это bool. `"False"` (в кавычках) — это str, просто текст
- F-строка без `f` — просто текст: `print("{name}")` выведет буквально `{name}`
- `input()` всегда возвращает строку, даже если ввели число. Нужно оборачивать в `int()` или `float()`
- Именуй переменные так, чтобы было понятно без комментариев: `free_disk_space_gb = 500` лучше чем `x = 500`
