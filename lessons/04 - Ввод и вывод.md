# Ввод и вывод

> print() показывает данные, input() запрашивает. Без них программа — закрытая коробка.

## Theory

- `print()` — выводит данные на экран
- `input("подсказка")` — запрашивает ввод, возвращает **строку**
- F-строка: `f"текст {переменная}"` — вставляет переменные

Форматирование чисел в f-строках:
- `{temp:.0f}` — без дробной части
- `{temp:.1f}` — один знак после запятой
- `{temp:.2f}` — два знака
- `{pct:.0%}` — проценты (автоматически ×100)
- `{text:<15}` — выравнивание влево, ширина 15
- `{text:>8}` — выравнивание вправо, ширина 8

## Code

```python
# Базовый вывод
print("Hello, World!")
print("RAM:", 16, "GB")  # автоматические пробелы

# Пустая строка
print("=== PC Booster ===")
print()
print("Версия 1.0")

# F-строки
name = "Lena"
ram_total = 16
ram_used = 11
print(f"Привет, {name}!")
print(f"Свободно RAM: {ram_total - ram_used} GB")  # можно выражения
print(f"Температура: {72.456:.1f}°C")  # → 72.5

# Форматирование чисел
used_pct = 0.671
print(f"Занято: {used_pct:.1%}")  # → 67.1%

# Ввод данных
name = input("Как тебя зовут? ")
print(f"Привет, {name}!")

# input() возвращает строку! Нужна конвертация
ram_gb = int(input("Введи RAM (GB): "))
print(f"RAM: {ram_gb} GB")

# Дробное число
temp = float(input("Температура: "))
print(f"Температура: {temp:.1f}°C")
```

## Practice

1. Напиши приветствие: спроси имя и город, выведи: `"Привет, {name}! Ты из {city}"`
2. Калькулятор памяти: спроси всего RAM и занято, выведи свободно в GB и %
3. Спроси количество ядер и частоту (GHz), посчитай производительность: ядра × частота

## Answers

```python
# 1
name = input("Имя: ")
city = input("Город: ")
print(f"Привет, {name}! Ты из {city}")

# 2
total = int(input("Всего RAM (GB): "))
used = int(input("Занято (GB): "))
free = total - used
pct = used / total
print(f"Свободно: {free} GB ({(1 - pct):.1%})")

# 3
cores = int(input("Количество ядер: "))
speed = float(input("Частота (GHz): "))
print(f"Производительность ≈ {cores * speed:.1f} GHz")
```

## Tips

- Забыл `f` перед строкой: `print("Город: {name}")` выведет буквально `{name}`, а не значение
- `input()` всегда возвращает строку: `ram + 4` → TypeError. Оборачивай в `int()` или `float()`
- `int()` — когда нужно целое, `float()` — когда нужна дробная часть
- Пустая строка `print()` — удобный разделитель блоков текста
- В `{}` можно писать выражения: `f"{x + y}"`, `f"{int(temp)}°C"`
