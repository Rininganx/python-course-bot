## 📖 Шаг 1 — Что такое вывод и ввод

Любая программа общается с пользователем двумя способами:

- **Вывод** (`print`) — программа показывает что-то на экране
- **Ввод** (`input`) — программа спрашивает данные у пользователя

Без этих двух — программа закрытая коробка. Весь текстовый интерфейс твоего будущего PC Booster строится на них.

## 📖 Шаг 2 — print() — вывод на экран

### Самый простой случай

```python
print("Hello, World!")
```

Вывод:
```
Hello, World!
```

Ты передаёшь `print()` что выводить — она выводит это и переходит на новую строку.

────────────────────
### Выводить несколько значений

```python
print("RAM:", 16, "GB")
```

Вывод:
```
RAM: 16 GB
```

Между значениями автоматически вставляется пробел. Очень удобно для вывода информации.

**Ещё примеры:**

```python
city      = "Berlin"
temp      = 72.5
processes = 42

print("Город:", city)
print("Температура:", temp)
print("Процессов запущено:", processes)
```

Вывод:
```
Город: Berlin
Температура: 72.5
Процессов запущено: 42
```

────────────────────
### Пустая строка

```python
print("=== PC Booster ===")
print()                        # пустая строка
print("Версия 1.0")
```

Вывод:
```
=== PC Booster ===

Версия 1.0
```

Удобно для разделения блоков текста.

## 📖 Шаг 3 — F-строки (форматирование)

F-строка — это строка с буквой `f` перед кавычками. Внутри фигурных скобок `{}` можно писать имена переменных, и они подставятся:

```python
name = "Lena"
city = "Berlin"

print(f"Привет, {name}!")
print(f"Ты из города {city}")
```

Вывод:
```
Привет, Lena!
Ты из города Berlin
```

**Это намного удобнее чем:**

```python
# Старый способ — конкатенация, очень неудобно
print("Привет, " + name + "!")

# Или совсем старый способ
print("Привет, {}!".format(name))
```

**Всегда используй f-строки** — современный и читаемый способ.

────────────────────
### Выражения внутри f-строки

В `{}` можно писать не только переменные, но и выражения:

```python
ram_total = 16
ram_used  = 11

print(f"Свободно RAM: {ram_total - ram_used} GB")
# → Свободно RAM: 5 GB

x = 10
y = 20
print(f"Сумма: {x + y}")
# → Сумма: 30

temp = 75.456789
print(f"Температура: {int(temp)}°C")
# → Температура: 75°C
```

## 📖 Шаг 4 — Форматирование чисел

В f-строке после двоеточия можно указать как форматировать число.

### Количество знаков после точки

```python
temp = 72.456789

print(f"Температура: {temp:.0f}°C")    # без дробной части
# → Температура: 72°C

print(f"Температура: {temp:.1f}°C")    # один знак
# → Температура: 72.5°C

print(f"Температура: {temp:.2f}°C")    # два знака
# → Температура: 72.46°C
```

### Проценты

```python
used_pct = 0.671234

print(f"Занято: {used_pct:.0%}")   # → Занято: 67%
print(f"Занято: {used_pct:.1%}")   # → Занято: 67.1%
```

Форматирование `%` автоматически умножает на 100.

### Выравнивание для таблиц

```python
print(f"{'Процесс':<15} {'RAM':>8}")
print(f"{'chrome.exe':<15} {850:>6} MB")
print(f"{'discord.exe':<15} {220:>6} MB")
```

Вывод:
```
Процесс         RAM
chrome.exe       850 MB
discord.exe      220 MB
```

- `<15` — выровнять влево, ширина 15 символов
- `>8` — выровнять вправо, ширина 8 символов

────────────────────
### Таблица форматов

Формат • Пример • Результат
`:.0f` • `{72.9:.0f}` • `73`
`:.1f` • `{72.456:.1f}` • `72.5`
`:.2f` • `{72.456:.2f}` • `72.46`
`:.1%` • `{0.671:.1%}` • `67.1%`
`:<10` • `{'hi':<10}` • `hi        `
`:>10` • `{'hi':>10}` • `        hi`

## 📖 Шаг 5 — input() — получить данные от пользователя

### Базовое использование

```python
name = input("Введи своё имя: ")
print(f"Привет, {name}!")
```

Работа:
```
Введи своё имя: Lena
Привет, Lena!
```

`input()` выводит подсказку, ждёт пока пользователь напишет и нажмёт Enter, затем сохраняет введённое в переменную.

────────────────────
### Главное правило: input() возвращает СТРОКУ

Даже если пользователь ввёл число — это будет строка:

```python
ram = input("Введи RAM (GB): ")
print(type(ram))    # → <class 'str'>
print(ram)          # → 16  (это строка "16", не число 16)
```

Попытка сложить строку с числом — ошибка:

```python
ram = input("RAM: ")     # пользователь ввёл "16"
print(ram + 4)          # → TypeError!
```

────────────────────
### Конвертация в число

Чтобы получить число — оберни `input()` в `int()` или `float()`:

```python
# Целое число
ram_gb = int(input("Введи RAM (GB): "))
print(type(ram_gb))       # → <class 'int'>
print(ram_gb + 4)         # → 20  (если ввели 16)

# Дробное число
temp = float(input("Введи температуру: "))
print(f"Температура: {temp:.1f}°C")

# Несколько вводов подряд
cores = int(input("Количество ядер: "))
speed = float(input("Частота (GHz): "))
print(f"Производительность ≈ {cores * speed:.1f} GHz")
```

## Задание

Напиши программу для PC Booster, которая запрашивает у пользователя характеристики ПК и выводит красивый отчёт.

**Дано:**
- Пользователь вводит: имя ПК, количество ядер CPU, объём RAM (GB), объём SSD (GB)

**Нужно:**
1. Запросить все данные через `input()`
2. Посчитать: суммарную тактовую частоту (ядер × 3.5 GHz)
3. Вывести отчёт в рамке

**Ожидаемый вывод:**
```
╔═══════════════════════════════╗
║   PC Booster — Спецификация   ║
╠═══════════════════════════════╣
║ ПК:     MyPC                  ║
║ CPU:    8 ядер (28.0 GHz)     ║
║ RAM:    16 GB                 ║
║ SSD:    512 GB                ║
╚═══════════════════════════════╝
```

💡 Подсказка 1
> Используй `input()` для каждого параметра. Не забудь конвертировать числа через `int()`.

💡 Подсказка 2
> Суммарная частота = ядра × 3.5. Выведи через `f"{freq:.1f} GHz"`.

💡 Подсказка 3
> Для рамки используй символы `╔ ╗ ║ ╠ ═ ╝ ╚` — просто скопируй их из примера.

## Решение

```python
print("=== PC Booster — Спецификация ===")
print()

pc_name = input("Имя ПК: ")
cores   = int(input("Количество ядер CPU: "))
ram_gb  = int(input("Объём RAM (GB): "))
ssd_gb  = int(input("Объём SSD (GB): "))

total_ghz = cores * 3.5

print()
print("╔═══════════════════════════════╗")
print("║   PC Booster — Спецификация   ║")
print("╠═══════════════════════════════╣")
print(f"║ ПК:     {pc_name:<20}║")
print(f"║ CPU:    {cores} ядер ({total_ghz:.1f} GHz){' ' * max(0, 10 - len(str(cores)) - len(f'{total_ghz:.1f}'))}║")
print(f"║ RAM:    {ram_gb} GB{' ' * max(0, 20 - len(str(ram_gb)))}║")
print(f"║ SSD:    {ssd_gb} GB{' ' * max(0, 20 - len(str(ssd_gb)))}║")
print("╚═══════════════════════════════╝")
```

**Разбор:**
1. `input("Имя ПК: ")` → спрашиваем имя, сразу сохраняем в переменную
2. `int(input(...))` → оборачиваем `input()` в `int()`, чтобы получить число, а не строку
3. `cores * 3.5` → умножаем ядра на типичную частоту одного ядра → `28.0`
4. `f"{total_ghz:.1f}"` → `.1f` означает один знак после запятой → `28.0`
5. Рамка собирается из отдельных `print()` — каждая строка это отдельная линия

💡 Упрощённый вариант (без рамки)
> Если рамка кажется сложной — начни с простого вывода:
> ```python
> print(f"ПК: {pc_name}")
> print(f"CPU: {cores} ядер ({total_ghz:.1f} GHz)")
> print(f"RAM: {ram_gb} GB")
> print(f"SSD: {ssd_gb} GB")
> ```
> Рамка — это просто красивая обёртка, суть та же.

## ❌ Частые ошибки

🐛 Забыл f перед строкой
> ```python
> name = "Berlin"
> print("Город: {name}")    # → Город: {name}  (буквально!)
> print(f"Город: {name}")   # → Город: Berlin  ✓
> ```

🐛 input() возвращает строку, не число
> ```python
> age = input("Возраст: ")  # "25" — строка!
> print(age + 10)           # TypeError!
>
> age = int(input("Возраст: "))  # ✓ теперь число
> print(age + 10)                # 35
> ```

🐛 Забыл двоеточие в f-строке для формата
> ```python
> temp = 72.456
> print(f"Темп: {temp.1f}°C")   # SyntaxError — неправильный синтаксис
> print(f"Темп: {temp:.1f}°C")  # ✓
> ```

## 💡 Заметки на полях

💡 F-строки — современный питоновский стиль
> Забудь про конкатенацию (`"text" + var + "text"`), забудь про `.format()`. F-строки — единственный способ который используют в настоящем коде.

💡 `int()` vs `float()`
> - `int()` — когда нужно целое число (количество, индекс)
> - `float()` — когда нужна дробная часть (температура, проценты, деньги)
