# Строки

> Строка — последовательность символов. Любой текст в Python — это строка: имя, путь, сообщение.

## Theory

Создание строки: кавычки двойные или одинарные, разницы нет.
Длина: `len(s)` — количество символов.
Индексы: нумерация с 0. Отрицательные — с конца: `s[-1]` последний символ.
Срезы: `s[начало:конец]` — конец не входит. `s[:5]` — с начала, `s[5:]` — до конца.
Строки неизменяемы! Методы возвращают новую строку, не меняют оригинал.

Спецсимволы:
- `\n` — перенос строки
- `\t` — табуляция
- `\\` — обратный слеш
- `r"путь"` — raw-строка (для Windows путей)

## Code

```python
city = "Berlin"
print(len(city))      # 9
print(city[0])        # B
print(city[-1])       # n (последний)
print(city[0:4])      # Berl

# Практические срезы
filename = "report_2026-05-15.txt"
print(filename[-4:])   # .txt
print(filename[:-4])   # report_2026-05-15
print(filename[::-1])  # строка наоборот

# Регистр
s = "berlin"
print(s.upper())       # BERLIN
print(s.capitalize())  # Berlin

# Пробелы
raw = "   Lena   "
print(raw.strip())     # "Lena"

# Поиск
email = "lena@berlin.de"
print("berlin" in email)          # True
print(email.startswith("lena"))   # True
print(email.find("@"))           # 4

# Замена
path = "C:/Users/Lena/file.txt"
print(path.replace("/", "\\"))    # C:\Users\Lena\file.txt

# Разделение и объединение
csv = "a,b,c"
print(csv.split(","))            # ['a', 'b', 'c']
print(", ".join(["x", "y"]))     # x, y

# Цепочки методов
raw = "   LENA SCHMIDT   "
result = raw.strip().lower().replace(" ", "_")
print(result)  # lena_schmidt
```

## Practice

1. Дано: `"  Chrome.EXE  "`. Убери пробелы, переведи в нижний, выведи.
2. Дан путь `"C:/Users/Lena/report.txt"`. Извлеки имя файла (после последнего `/`).
3. Дан email `"  Lena@BERLIN.DE  "`. Очисти и нормализуй (strip + lower).
4. Дан лог: `"ERROR|Berlin|42"`. Разбей по `|` и выведи каждое поле.

## Answers

```python

# 2

print(f"Тип: {parts[0]}")

print(f"Код: {parts[2]}")
```

## Tips

- Строки неизменяемы! `name.upper()` вернёт результат, но `name` останется `"lena"`. Сохраняй: `name = name.upper()`
- `find()` возвращает -1 если не найдено. Лучше проверяй через `in`: `"x" in s`
- `r"путь"` — raw-строка для Windows путей. Без `r` символы `\U`, `\t` могут сломать строку
- `rfind(".")` — найти последнюю точку. Полезно для расширений файлов: `"my.report.txt".rfind(".")` → 9
- Пробел тоже символ: `len("PC Booster")` → 10, не 9
