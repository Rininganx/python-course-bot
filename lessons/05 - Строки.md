## 📖 Шаг 1 — Что такое строка

Строка (`str`) — это последовательность символов. Любой текст в Python — это строка: имя города, путь к файлу, название процесса, сообщение об ошибке.

```python
city     = "Berlin"
filename = "report.txt"
path     = r"C:\Users\Lena\Desktop"

print(city)      # → Berlin
print(filename)  # → report.txt
print(path)      # → C:\Users\Lena\Desktop
```

Строку обозначают кавычками — двойными или одинарными, разницы нет:

```python
s1 = "Berlin"
s2 = 'Berlin'
print(s1 == s2)  # → True  (одинаковые)
```

## 📖 Шаг 2 — Спецсимволы

Некоторые символы нельзя вставить напрямую. Используй обратный слеш `\`:

```python
# \n — перенос строки
print("Berlin\nMunich\nVienna")
# → Berlin
# → Munich
# → Vienna

# \t — табуляция (отступ)
print("Город:\tBerlin")
print("Население:\t3.6 млн")
# → Город:        Berlin
# → Население:    3.6 млн

# \\ — обратный слеш
print("Путь: C:\\Users\\Lena")
# → Путь: C:\Users\Lena

# \" — двойная кавычка
print("Девиз: \"Быстрее, выше, сильнее\"")
# → Девиз: "Быстрее, выше, сильнее"
```

### Raw-строки для Windows путей

Если в строке много обратных слешей — поставь `r` перед кавычками:

```python
# БЕЗ r — \U воспринимается как спецсимвол → ошибка
path = "C:\Users\Lena\temp"  # проблема!

# С r — всё воспринимается буквально
path = r"C:\Users\Lena\temp"
print(path)   # → C:\Users\Lena\temp  ✓
```

💡 Правило для Windows путей
> Всегда пиши пути как `r"C:\Users\..."` — избавит от неожиданных ошибок.

## 📖 Шаг 4 — Индексы — обращение к символам

Каждый символ строки имеет номер — индекс. Нумерация начинается с **нуля**:

```python
s = "Berlin"
#    012345

print(s[0])    # → B
print(s[1])    # → e
print(s[2])    # → r
print(s[5])    # → n  (последний)
```

**Отрицательные индексы** считают с конца:

```python
s = "Berlin"
#    -6-5-4-3-2-1

print(s[-1])   # → n  (последний)
print(s[-2])   # → i
print(s[-6])   # → B  (первый)
```

Визуализация:

```
  B    e    r    l    i    n
  0    1    2    3    4    5
 -6   -5   -4   -3   -2   -1
```

⚠️ IndexError — выход за пределы
> ```python
> s = "Lena"
> print(s[10])   # IndexError — нет такого индекса
> ```
> Максимальный индекс: `len(s) - 1`

## 📖 Шаг 5 — Срезы (Slicing)

Срез позволяет взять **часть** строки. Синтаксис: `s[начало:конец]`

```python
s = "Amsterdam"
#    012345678

print(s[0:5])    # → Amste  (с 0 по 4 включительно, 5 не входит)
print(s[5:9])    # → rdam
print(s[:5])     # → Amste  (с начала до 5)
print(s[5:])     # → rdam   (с 5 до конца)
print(s[:])      # → Amsterdam  (вся строка)
```

### Практические срезы

```python
filename = "report_2026-05-15.txt"

# Расширение — последние 4 символа
ext = filename[-4:]
print(ext)           # → .txt

# Имя без расширения
name = filename[:-4]
print(name)          # → report_2026-05-15

# Дата — символы 7–17
date = filename[7:17]
print(date)          # → 2026-05-15

# Строка наоборот
word = "Guten"
print(word[::-1])    # → netuG
```

## 📖 Шаг 6 — Методы строк

Метод — это функция встроенная в объект. Вызывается через точку: `строка.метод()`.

⚠️ Строки неизменяемы!
> Методы **не меняют** оригинальную строку — они возвращают **новую**.
> ```python
> name = "lena"
> name.upper()        # вернул "LENA", но name осталась "lena"
> name = name.upper() # теперь name = "LENA"
> ```

────────────────────
### Регистр

```python
city = "berlin"

print(city.upper())        # → BERLIN
print(city.lower())        # → berlin
print(city.capitalize())   # → Berlin  (первая большая, остальные маленькие)
print(city.title())        # → Berlin

# Практика: нормализация имён городов
cities = ["berlin", "MUNICH", "vIEnNA"]
for c in cities:
    print(c.title())
# → Berlin
# → Munich
# → Vienna
```

### Удаление пробелов

```python
raw = "   Lena Schmidt   "

print(raw.strip())     # → "Lena Schmidt"  (оба конца)
print(raw.lstrip())    # → "Lena Schmidt   "  (только слева)
print(raw.rstrip())    # → "   Lena Schmidt"  (только справа)

# Часто — очистить ввод пользователя
user_input = input("Введи имя: ").strip()
```

### Поиск и проверка

```python
email = "lena.schmidt@berlin.de"

# in — самый питоновский способ
print("berlin" in email)       # → True
print("munich" in email)       # → False

# startswith / endswith
print(email.startswith("lena"))   # → True
print(email.endswith(".de"))      # → True
print(email.endswith(".com"))     # → False

# find — позиция (или -1 если не найдено)
print(email.find("@"))           # → 12
print(email.find("malware"))     # → -1

# Подсчёт вхождений
log = "ERROR Berlin ERROR Munich ERROR Vienna"
print(log.count("ERROR"))        # → 3
```

### Разделение и объединение

```python
# split() — строка → список
csv = "Berlin,Munich,Vienna,Amsterdam"
cities = csv.split(",")
print(cities)
# → ['Berlin', 'Munich', 'Vienna', 'Amsterdam']

# split() без аргумента — по пробелам
sentence = "Guten  Morgen   Berlin"
words = sentence.split()
print(words)
# → ['Guten', 'Morgen', 'Berlin']

# join() — список → строка
cities = ["Berlin", "Munich", "Vienna"]
result = ", ".join(cities)
print(result)   # → Berlin, Munich, Vienna

# Дата из частей
parts = ["2026", "05", "15"]
date = "-".join(parts)
print(date)   # → 2026-05-15
```

## 📖 Шаг 7 — Цепочки методов

Методы можно вызывать один за другим:

```python
raw = "   LENA SCHMIDT   "

# Шаг за шагом
step1 = raw.strip()           # "LENA SCHMIDT"
step2 = step1.lower()         # "lena schmidt"
step3 = step2.replace(" ", "_")  # "lena_schmidt"
print(step3)

# Всё в одну строку — цепочка
result = raw.strip().lower().replace(" ", "_")
print(result)   # → lena_schmidt

# Нормализация email
email = "  Lena.Schmidt@BERLIN.DE  "
normalized = email.strip().lower()
print(normalized)   # → lena.schmidt@berlin.de
```

filename = "  SystemReport_2026-05-08.exe  "

# Очистить и нормализовать
clean = filename.strip().lower()

    ext  = "(нет расширения)"

print(f"Имя файла : {filename}")
print(f"Имя       : {name}")
print(f"Расширение: {ext}")
print(f"Символов  : {len(clean)}")

## Задание

Напиши функцию `clean_process_name(raw_name)` для PC Booster, которая очищает и нормализует имя процесса.

**Дано:**
- Сырое имя процесса: `"  Chrome.EXE  "`

**Нужно:**
1. Убрать пробелы по краям
2. Перевести в нижний регистр
3. Проверить заканчивается ли на `.exe`
4. Вернуть словарь с результатом

**Ожидаемый вывод:**
```
Оригинал: '  Chrome.EXE  '
Очищено:  'chrome.exe'
Длина:    10 символов
Это .exe?: Да
```

💡 Подсказка 1
> Используй `.strip()` для удаления пробелов и `.lower()` для нижнего регистра.

💡 Подсказка 2
> Проверку расширения можно сделать через `.endswith(".exe")` — вернёт `True` или `False`.

💡 Подсказка 3
> Чтобы вывести `Да`/`Нет` из `True`/`False`, используй тернарный оператор:
> `"Да" if is_exe else "Нет"`

## Решение

```python
def clean_process_name(raw_name: str) -> dict:
    """Очищает и нормализует имя процесса."""
    cleaned = raw_name.strip().lower()
    is_exe = cleaned.endswith(".exe")

    return {
        "original":  raw_name,
        "cleaned":   cleaned,
        "length":    len(cleaned),
        "is_exe":    is_exe,
    }

# Тест
result = clean_process_name("  Chrome.EXE  ")

print(f"Оригинал: '{result['original']}'")
print(f"Очищено:  '{result['cleaned']}'")
print(f"Длина:    {result['length']} символов")
print(f"Это .exe?: {'Да' if result['is_exe'] else 'Нет'}")
```

**Разбор:**
1. `raw_name.strip()` → убирает пробелы по краям → `"Chrome.EXE"`
2. `.lower()` → переводит в нижний регистр → `"chrome.exe"`
3. `.endswith(".exe")` → проверяет окончание строки → `True`
4. `len(cleaned)` → считает количество символов → `10`
5. `"Да" if result['is_exe'] else "Нет"` → тернарный оператор превращает `True` в `"Да"`

## ❌ Частые ошибки

🐛 Забыл что строки неизменяемы
> ```python
> name = "lena"
> name.upper()    # результат потерян!
> print(name)     # → lena  (не изменилась)
>
> name = name.upper()  # ✓ сохранить результат
> print(name)     # → LENA
> ```

🐛 find() возвращает -1 если не найдено
> ```python
> s = "berlin"
> pos = s.find("x")
> print(pos)    # → -1  (это не индекс!)
>
> # Лучше проверять через in:
> if "x" in s:
>     ...
> ```

🐛 IndexError — выход за пределы
> ```python
> s = "Lena"
> print(s[10])   # IndexError!
>
> # Проверяй: индекс < len(s)
> if i < len(s):
>     print(s[i])
> ```

## 💡 Заметки на полях

💡 rfind() — найти последнее вхождение
> ```python
> filename = "my.report.2026.txt"
> dot = filename.rfind(".")      # последняя точка
> ext = filename[dot:]
