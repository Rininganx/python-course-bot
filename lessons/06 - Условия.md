# Условия (if / elif / else)

> Программа учится принимать решения — как навигатор с развилкой.

## Theory

Условие — развилка в коде. Если условие истинно — выполняется блок, иначе — пропускается.

`if` — проверяет условие. `elif` — проверяет следующее, если предыдущее ложно. `else` — выполняется, если все условия ложны.

Python проверяет условия **сверху вниз** и останавливается на первом `True`. Порядок важен — сначала строгие условия.

Отступы (4 пробела) — обязательны, без них будет `IndentationError`. Количество `elif` не ограничено.

Тернарный оператор: `значение_A if условие else значение_B` — заменяет короткий if/else в одну строку.

## Code

```python
# Базовый if
temp = 85
if temp > 90:
    print("Перегрев!")

# if / else
if temp > 90:
    print("Перегрев!")
else:
    print("Норма.")

# if / elif / else — от строгого к мягкому
if temp >= 95:
    print("КРИТИЧНО")
elif temp >= 80:
    print("ВНИМАНИЕ")
elif temp >= 60:
    print("НОРМА")
else:
    print("ХОЛОДНО")

# Тернарный оператор
label = "горячо" if temp > 80 else "норма"

# Truthiness — что Python считает False
print(bool(0))      # False
print(bool(""))     # False
print(bool([]))     # False
print(bool(42))     # True
print(bool("да"))   # True

# Вложенные условия
ram = 88
cpu = 91
if cpu > 90:
    if ram > 85:
        print("Критическая ситуация!")
    else:
        print("CPU горячий, но RAM держится.")

# pass — пустой блок
if temp > 100:
    pass  # TODO: добавить логику
```

## Practice

1. Напиши программу: ввод температуры CPU. Если >= 95 → "КРИТИЧНО", >= 80 → "ВНИМАНИЕ", >= 60 → "НОРМА", иначе → "ХОЛОДНО".
2. Создай переменную `free_gb = 0`. Выведи "Места нет" если она равна 0, иначе "Место есть".
3. Используя тернарный оператор, определи: `age = 16` → "подросток" если < 18, иначе "взрослый".
4. Дан `score = 75`. Вложенные условия: если >= 90 → "отлично", иначе если >= 70 → "хорошо", иначе → "учиться".

## Answers

```python
# Задача 1
temp = float(input("Температура CPU: "))
if temp >= 95:
    print("КРИТИЧНО")
elif temp >= 80:
    print("ВНИМАНИЕ")
elif temp >= 60:
    print("НОРМА")
else:
    print("ХОЛОДНО")

# Задача 2
free_gb = 0
print("Места нет" if free_gb == 0 else "Место есть")

# Задача 3
age = 16
status = "подросток" if age < 18 else "взрослый"
print(status)

# Задача 4
score = 75
if score >= 90:
    print("отлично")
elif score >= 70:
    print("хорошо")
else:
    print("учиться")
```

## Tips

- `=` в условии — SyntaxError. Используй `==`.
- Порядок elif: сначала строгие (>= 95), потом мягкие (>= 60).
- Без отступа после `if` — ошибка. Всегда 4 пробела.
