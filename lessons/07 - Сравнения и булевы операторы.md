# Сравнения и булевы операторы

> Объединяй условия через and/or/not — одна строка вместо вложенных if.

## Theory

Операторы сравнения (`==`, `!=`, `>`, `<`, `>=`, `<=`) возвращают `True` или `False`.

Булевы операторы объединяют условия:
- `and` — оба должны быть True
- `or` — достаточно одного True
- `not` — инвертирует значение

`in` / `not in` — проверяют вхождение в строку или список. `is` — только для `None`.

Python использует **короткое замыкание**: `and` не проверяет второе условие, если первое ложно. `or` не проверяет второе, если первое истинно.

Приоритет: `not` > `and` > `or`. При смешивании используй скобки.

## Code

```python
# Сравнения
print(10 > 5)         # True
print("abc" == "ABC")  # False (регистр важен)
print(5 >= 5)          # True

# Сравнение строк — лексикографическое
print("Amsterdam" < "Berlin")  # True (A раньше B)

# and — оба True
temp, ram = 88, 92
print(temp > 80 and ram > 80)   # True
print(temp > 95 and ram > 80)   # False

# or — хотя бы одно True
print(temp > 95 or ram > 80)    # True
print(temp > 95 or ram > 95)    # False

# not — инверсия
is_running = True
print(not is_running)           # False

# in — вхождение в строку
filename = "setup.exe"
print(".exe" in filename)       # True
print("berlin" in filename)     # False

# in — проверка в списке
allowed = [".exe", ".msi", ".bat"]
print(".exe" in allowed)        # True

# is — только для None
result = None
if result is None:
    print("Данные не получены")

# Цепочки сравнений — компактная запись
age = 25
if 18 <= age < 65:
    print("рабочий возраст")

# Короткое замыкание
x = 0
if x != 0 and 100 / x > 5:
    print("ok")
else:
    print("безопасно — деление пропущено")
```

## Practice

1. Введи RAM (GB) и температуру CPU. Если RAM >= 16 И CPU < 80 → "Готов к игре!", иначе → "Не готов".
2. Проверь строку `filename = "report.pdf"`. Если расширение `.pdf` или `.docx` → "Документ", иначе → "Не документ".
3. Напиши цепочку: `score = 1850`. Если 1500 <= score < 2000 → "Silver", если >= 2000 → "Gold", иначе → "Bronze".
4. Проверь: `name = ""`. Если name не пустой и длиннее 2 символов → "ОК", иначе → "Ошибка".

## Answers

```python
# Задача 1
ram = int(input("RAM (GB): "))
temp = float(input("CPU temp: "))
print("Готов к игре!" if ram >= 16 and temp < 80 else "Не готов")

# Задача 2
filename = "report.pdf"
ext = filename.split(".")[-1]
print("Документ" if ext in ["pdf", "docx"] else "Не документ")

# Задача 3
score = 1850
if score >= 2000:
    tier = "Gold"
elif score >= 1500:
    tier = "Silver"
else:
    tier = "Bronze"
print(tier)

# Задача 4
name = ""
if name and len(name) > 2:
    print("ОК")
else:
    print("Ошибка")
```

## Tips

- `=` вместо `==` в условии — SyntaxError.
- `==` сравнивает значения, `is` сравнивает объект в памяти. Для `None` используй `is`.
- При смешивании `and`/`or` добавляй скобки для ясности.
