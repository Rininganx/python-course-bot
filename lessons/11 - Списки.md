## Главная идея

Список — это упорядоченная коллекция элементов. Может хранить что угодно: числа, строки, другие списки. Элементы можно менять, добавлять, удалять.

────────────────────
## Создание и доступ

```python
procs = ["chrome.exe", "discord.exe", "steam.exe"]

print(procs[0])    # → chrome.exe   (первый)
print(procs[-1])   # → steam.exe    (последний)
print(procs[1:])   # → ['discord.exe', 'steam.exe']
print(len(procs))  # → 3
```

## Методы списка

Метод • Что делает • Пример
`.append(x)` • Добавить в конец • `lst.append("new")`
`.insert(i, x)` • Вставить на позицию i • `lst.insert(0, "first")`
`.extend(lst2)` • Добавить все элементы другого списка • `lst.extend([1,2,3])`
`.remove(x)` • Удалить первый x • `lst.remove("chrome")`
`.pop(i)` • Удалить и вернуть элемент по индексу • `lst.pop(-1)`
`.pop()` • Удалить и вернуть последний • `lst.pop()`
`.clear()` • Очистить список • `lst.clear()`
`.sort()` • Сортировать по возрастанию • `lst.sort()`
`.sort(reverse=True)` • Сортировать по убыванию • `lst.sort(reverse=True)`
`.reverse()` • Перевернуть • `lst.reverse()`
`.index(x)` • Индекс первого x • `lst.index("chrome")`
`.count(x)` • Сколько раз встречается x • `lst.count("chrome")`
`.copy()` • Копия списка • `lst2 = lst.copy()`

```python
# sorted() — не меняет оригинал, возвращает новый
nums = [3, 1, 4, 1, 5]
print(sorted(nums))           # → [1, 1, 3, 4, 5]
print(sorted(nums, reverse=True))  # → [5, 4, 3, 1, 1]
print(nums)                   # → [3, 1, 4, 1, 5] — не изменился
```

## List comprehension — список одной строкой

```python
# Обычный способ
squares = []
for i in range(5):
    squares.append(i ** 2)

# List comprehension
squares = [i ** 2 for i in range(5)]
print(squares)  # → [0, 1, 4, 9, 16]

# С условием
big = [x for x in [100, 450, 820, 60, 910] if x > 400]
print(big)  # → [450, 820, 910]
```

## Практика — PC Booster

```python
# Управление списком процессов
processes = ["chrome.exe", "discord.exe", "svchost.exe", "steam.exe"]
memory    = [850, 220, 45, 340]

# Добавить новый процесс
processes.append("teams.exe")
memory.append(680)

# Удалить безопасный системный
idx = processes.index("svchost.exe")
processes.pop(idx)
memory.pop(idx)

# Отсортировать по памяти (вместе)
combined = list(zip(processes, memory))
combined.sort(key=lambda x: x[1], reverse=True)

print("=== Топ процессов по RAM ===")
for i, (name, mem) in enumerate(combined, 1):
    bar = "█" * (mem // 100)
    print(f"{i}. {name:<15} {mem:>5} MB  {bar}")
```

Вывод:
```
=== Топ процессов по RAM ===
1. chrome.exe       850 MB  ████████
2. teams.exe        680 MB  ██████
3. steam.exe        340 MB  ███
4. discord.exe      220 MB  ██
```

## Задание

PC Booster управляет автозагрузкой. Напиши программу которая фильтрует что отключить.

**Дано:**
```python
startup = ["OneDrive", "Spotify", "Discord", "Teams", "Steam", "Skype"]
safe_to_disable = ["Spotify", "Steam", "Skype", "OneDrive"]
```

**Нужно:**
1. Создать список `to_disable` — только те из `startup` которые есть в `safe_to_disable`
2. Создать список `keep` — остальные
3. Вывести оба списка
4. Посчитать сколько MB освободится (каждое отключённое = 50 MB)

**Ожидаемый вывод:**
```
Отключить (4): ['OneDrive', 'Spotify', 'Steam', 'Skype']
Оставить  (2): ['Discord', 'Teams']
Освободится: ~200 MB RAM
```

💡 Подсказка 1
> Используй list comprehension: `[app for app in startup if app in safe_to_disable]`

💡 Подсказка 2
> Для `keep` — аналогично, но с `not in`: `[app for app in startup if app not in safe_to_disable]`

💡 Подсказка 3
> Количество MB = `len(to_disable) * 50`

## Решение

```python
startup = ["OneDrive", "Spotify", "Discord", "Teams", "Steam", "Skype"]
safe_to_disable = ["Spotify", "Steam", "Skype", "OneDrive"]

to_disable = [app for app in startup if app in safe_to_disable]
keep = [app for app in startup if app not in safe_to_disable]

print(f"Отключить ({len(to_disable)}): {to_disable}")
print(f"Оставить  ({len(keep)}): {keep}")
print(f"Освободится: ~{len(to_disable) * 50} MB RAM")
```

**Разбор:**
1. `[app for app in startup if app in safe_to_disable]` → list comprehension: перебирает `startup`, оставляет только те что в `safe_to_disable`
2. `not in` → инверсия: оставляет только те что **не** в списке на отключение
3. `len(to_disable)` → количество элементов в списке → `4`
4. `4 * 50 = 200` → простая арифметика
5. List comprehension заменяет 4 строки (создать пустой список, цикл, if, append) одной строкой

## Заметки на полях

⚠️ Копирование списка
> ```python
> a = [1, 2, 3]
> b = a          # b — это ТОТ ЖЕ список!
> b.append(4)
> print(a)       # → [1, 2, 3, 4]  — a тоже изменился!
>
> b = a.copy()   # теперь b — независимая копия
> ```

💡 Проверка наличия элемента
> ```python
> if "chrome.exe" in processes:
>     print("Chrome запущен")
> ```
> Быстро, читаемо, без цикла.
