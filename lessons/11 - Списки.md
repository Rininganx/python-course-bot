# Списки

> Упорядоченная коллекция: числа, строки, другие списки. Элементы можно менять.

## Theory

Список — изменяемая последовательность. Хранит что угодно, доступ по индексу `[0]`, `[-1]` для последнего. Срезы `[1:]`, вложенность друг в друга.

## Code

```python
# Создание
procs = ["chrome.exe", "discord.exe", "steam.exe"]
print(procs[0])    # → chrome.exe
print(procs[-1])   # → steam.exe
print(len(procs))  # → 3

# Методы
procs.append("teams.exe")      # добавить в конец
procs.insert(0, "first")       # вставить в начало
procs.remove("chrome.exe")     # удалить по значению
procs.pop()                    # удалить и вернуть последний
procs.sort()                   # сортировка
procs.reverse()                # перевернуть
procs.copy()                   # копия

# sorted() — не меняет оригинал
nums = [3, 1, 4, 1, 5]
print(sorted(nums))  # → [1, 1, 3, 4, 5]
print(nums)          # → [3, 1, 4, 1, 5]

# List comprehension
squares = [i ** 2 for i in range(5)]  # → [0, 1, 4, 9, 16]
big = [x for x in [100, 450, 820] if x > 400]  # → [450, 820]
```

## Practice

1. Создай список процессов, отсортируй по имени, выведи топ-3
2. Напиши comprehension: только чётные числа из `[1,2,3,4,5,6,7,8]`

## Answers

```python
# 1
procs = ["steam.exe", "chrome.exe", "teams.exe", "discord.exe"]
procs.sort()
print(procs[:3])

# 2
evens = [x for x in range(1, 9) if x % 2 == 0]
print(evens)  # → [2, 4, 6, 8]
```

## Tips

- `a = b` — это ТА ЖЕ ссылка. Используй `a = b.copy()`
- `"x" in lst` — быстрая проверка наличия
- `lst.append(x)` добавляет один элемент, `lst.extend(lst2)` — все из другого списка
