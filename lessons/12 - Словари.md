# Словари

> Пары ключ → значение. Вместо индексов — имена.

## Theory

Словарь хранит данные как `ключ: значение`. Доступ по ключу `d["key"]` или безопасно `d.get("key", default)`. Ключи должны быть хешируемыми (строки, числа, кортежи).

## Code

```python
# Создание
system = {
    "name":   "MyPC",
    "ram_gb": 16,
    "cpu":    "Intel i7",
    "temp":   72.5
}

print(system["name"])          # → MyPC
print(system.get("gpu", "N/A")) # → N/A (безопасный доступ)

# Методы
system["gpu"] = "RTX 3080"     # добавить/изменить
del system["temp"]              # удалить
print(system.keys())            # все ключи
print(system.values())          # все значения
print(system.items())           # пары (ключ, значение)

# Итерация
for key, value in system.items():
    print(f"{key}: {value}")

# Вложенные словари
processes = {
    "chrome.exe": {"ram": 850, "cpu": 12.5},
    "discord.exe": {"ram": 220, "cpu": 3.1},
}
print(processes["chrome.exe"]["ram"])  # → 850

# Dict comprehension
names = ["chrome", "discord", "steam"]
mems = [850, 220, 340]
proc_dict = {name: mem for name, mem in zip(names, mems)}
heavy = {k: v for k, v in proc_dict.items() if v > 300}
```

## Practice

1. Создай словарь процессов и найди самый тяжёлый через `max()`
2. Посчитай суммарную RAM всех процессов через `sum()`

## Answers

```python
processes = {"chrome.exe": 850, "teams.exe": 680, "steam.exe": 340}

# 1
heaviest = max(processes, key=processes.get)
print(f"Самый тяжёлый: {heaviest} ({processes[heaviest]} MB)")

# 2
total = sum(processes.values())
print(f"Всего: {total} MB")
```

## Tips

- `d[key]` бросит `KeyError` если ключа нет, `d.get(key)` вернёт `None`
- Кортежи можно использовать как ключи, списки — нельзя
- `key in d` — быстрая проверка наличия ключа
