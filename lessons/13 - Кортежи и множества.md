# Кортежи и множества

> Кортеж — неизменяемый список. Множество — уникальные элементы без порядка.

## Theory

Кортеж `(tuple)` — как список, но нельзя менять. Используют для констант и возврата нескольких значений. Множество `(set)` — коллекция уникальных элементов с операциями: пересечение `&`, объединение `|`, разность `-`.

## Code

```python
# Кортеж
point = (10, 20)
process = ("chrome.exe", 850, 12.5)
single = (42,)  # запятая для одного элемента

# Распаковка
name, mem, cpu = process
print(name)  # → chrome.exe

# Обмен без temp
a, b = 1, 2
a, b = b, a  # → 2, 1

# Кортеж как ключ словаря
screen_res = {
    (1920, 1080): "Full HD",
    (2560, 1440): "2K",
}

# Множество
s = {1, 2, 3, 3, 2}  # → {1, 2, 3} — дубли удалены
s2 = set([1, 2, 2, 3])
empty = set()  # пустое (не {}, это словарь!)

# Операции
a = {"chrome", "discord", "steam"}
b = {"steam", "teams", "discord"}
print(a & b)  # пересечение → {'discord', 'steam'}
print(a | b)  # объединение → {'chrome', 'discord', 'steam', 'teams'}
print(a - b)  # разность → {'chrome'}

# Методы
s.add(4)       # добавить
s.remove(1)    # удалить (ошибка если нет)
s.discard(1)   # удалить (без ошибки)
x in s         # проверка наличия — быстро
```

## Practice

1. Убери дубли из списка через `set()`, сохрани порядок через `dict.fromkeys()`
2. Найди общие элементы двух списков через множества

## Answers

```python
# 1
lst = [3, 1, 2, 1, 3, 4, 2]
unique_ordered = list(dict.fromkeys(lst))
print(unique_ordered)  # → [3, 1, 2, 4]

# 2
a = [1, 2, 3, 4]
b = [3, 4, 5, 6]
common = set(a) & set(b)
print(common)  # → {3, 4}
```

## Tips

- Кортеж быстрее списка для неизменяемых данных
- Порядок в `set` не гарантирован
- Пустое множество: `set()`, не `{}` — это словарь
