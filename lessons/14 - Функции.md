## Главная идея

Функция — именованный блок кода который можно вызывать много раз. Пишешь логику один раз → используешь везде. Это основа чистого, читаемого кода.

## Параметры и аргументы

```python
# Позиционные аргументы
def add(a, b):
    return a + b

print(add(3, 5))     # → 8

# Дефолтные значения
def get_status(temp, threshold=80):
    return "горячо" if temp > threshold else "норма"

print(get_status(85))        # → горячо  (threshold=80 по умолчанию)
print(get_status(85, 90))    # → норма   (threshold=90)

# Именованные аргументы
print(get_status(temp=75, threshold=70))  # → горячо

# *args — любое количество аргументов
def total(*values):
    return sum(values)

print(total(100, 200, 300))   # → 600

# **kwargs — именованные аргументы как словарь
def show_info(**data):
    for key, val in data.items():
        print(f"{key}: {val}")

show_info(ram=16, cpu="i7", temp=72.5)
```

## Возврат нескольких значений

```python
def analyze_ram(total_gb, used_gb):
    free_gb  = total_gb - used_gb
    used_pct = used_gb / total_gb * 100
    free_pct = 100 - used_pct
    return free_gb, used_pct, free_pct   # возвращает кортеж

free, used_p, free_p = analyze_ram(16, 11)
print(f"Свободно: {free} GB ({free_p:.1f}%)")
print(f"Занято:   {16-free} GB ({used_p:.1f}%)")
```

## Практика — PC Booster

```python
def check_temp(temp: float) -> str:
    """Проверяет температуру CPU и возвращает статус."""
    if temp >= 95:
        return "КРИТИЧНО"
    elif temp >= 80:
        return "ВНИМАНИЕ"
    elif temp >= 60:
        return "НОРМА"
    else:
        return "ХОЛОДНО"

def ram_usage(total_gb: int, used_gb: int) -> dict:
    """Возвращает статистику использования RAM."""
    free    = total_gb - used_gb
    pct     = used_gb / total_gb * 100
    status  = "ОК" if pct < 80 else "ПЕРЕГРУЗКА"
    return {
        "total": total_gb,
        "used":  used_gb,
        "free":  free,
        "pct":   round(pct, 1),
        "status": status,
    }

def system_report(pc_name: str, temp: float, ram_total: int, ram_used: int):
    """Полный отчёт о системе."""
    ram = ram_usage(ram_total, ram_used)

    print(f"\n{'='*35}")
    print(f"  PC Booster — {pc_name}")
    print(f"{'='*35}")
    print(f"  CPU: {temp}°C [{check_temp(temp)}]")
    print(f"  RAM: {ram['used']}/{ram['total']} GB ({ram['pct']}%) [{ram['status']}]")
    print(f"  Свободно: {ram['free']} GB")
    print(f"{'='*35}")

system_report("MyPC", 87.5, 16, 13)
```

Вывод:
```
===================================
  PC Booster — MyPC
===================================
  CPU: 87.5°C [ВНИМАНИЕ]
  RAM: 13/16 GB (81.2%) [ПЕРЕГРУЗКА]
  Свободно: 3 GB
===================================
```

## Задание

PC Booster использует функции для анализа дисков. Напиши три функции.

**Дано:**
- `calc_free_space(total_mb, used_mb)` — возвращает кортеж `(free_mb, free_pct)`
- `get_recommendation(free_pct)` — возвращает строку-рекомендацию
- `disk_report(drive, total_mb, used_mb)` — использует обе функции и выводит отчёт

**Правила рекомендаций:**
- `< 10%` → `"Критически мало места — очистите диск"`
- `< 25%` → `"Рекомендуется очистка"`
- иначе → `"Место в норме"`

**Ожидаемый вывод:**
```
Диск C:
  Всего:     500000 MB
  Занято:    423000 MB
  Свободно:   77000 MB (15.4%)
  Статус: Рекомендуется очистка
```

💡 Подсказка 1
> `free_mb = total_mb - used_mb`, `free_pct = free_mb / total_mb * 100`. Верни оба через `return free_mb, free_pct`.

💡 Подсказка 2
> `get_recommendation` — чистая функция: принимает число, возвращает строку. Используй `if/elif/else`.

💡 Подсказка 3
> `disk_report` вызывает обе функции и собирает вывод. Не забудь `round(free_pct, 1)` для красивого процента.

## Решение

```python
def calc_free_space(total_mb: int, used_mb: int) -> tuple:
    """Возвращает свободное место в MB и процент."""
    free_mb  = total_mb - used_mb
    free_pct = free_mb / total_mb * 100
    return free_mb, free_pct

def get_recommendation(free_pct: float) -> str:
    """Возвращает рекомендацию по свободному месту."""
    if free_pct < 10:
        return "Критически мало места — очистите диск"
    elif free_pct < 25:
        return "Рекомендуется очистка"
    else:
        return "Место в норме"

def disk_report(drive: str, total_mb: int, used_mb: int):
    """Выводит полный отчёт о диске."""
    free_mb, free_pct = calc_free_space(total_mb, used_mb)
    status = get_recommendation(free_pct)

    print(f"Диск {drive}:")
    print(f"  Всего:     {total_mb} MB")
    print(f"  Занято:    {used_mb} MB")
    print(f"  Свободно:   {free_mb} MB ({free_pct:.1f}%)")
    print(f"  Статус: {status}")

disk_report("C:", 500000, 423000)
```

**Разбор:**
1. `calc_free_space` возвращает **кортеж** `(free_mb, free_pct)` — Python распакует его при вызове
2. `get_recommendation` — **чистая функция**: не печатает, не меняет глобальные переменные, только возвращает строку
3. `disk_report` — **оркестратор**: вызывает обе функции, собирает результат, выводит
4. `free_pct:.1f` → форматирование: один знак после запятой → `15.4`
5. Каждая функция делает **одну вещь** — это принцип единственной ответственности

## Заметки на полях

💡 Type hints — подсказки типов
> ```python
> def add(a: int, b: int) -> int:
> ```
> Python их не проверяет, но IDE подсвечивает ошибки и код читается лучше.

💡 Docstring — документируй функции
> ```python
> def my_func(x):
>     """Что делает функция, что принимает, что возвращает."""
>     ...
> help(my_func)   # выведет docstring
> ```

⚠️ Изменяемый дефолтный аргумент — ловушка
> ```python
> def add_item(item, lst=[]):   # ПЛОХО — список создаётся один раз!
>     lst.append(item)
>     return lst
>
> def add_item(item, lst=None): # ХОРОШО
>     if lst is None:
>         lst = []
>     lst.append(item)
>     return lst
> ```
