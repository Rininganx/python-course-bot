# Функции

> Именованный блок кода. Пишешь один раз — используешь везде.

## Theory

Функция определяется через `def`. Принимает параметры, может возвращать значение через `return`. Без `return` возвращает `None`. Параметры бывают: позиционные, с дефолтами, `*args` (кортеж), `**kwargs` (словарь).

## Code

```python
# Базовый синтаксис
def greet(name):
    return f"Привет, {name}!"

print(greet("MyPC"))  # → Привет, MyPC!

# Дефолтные значения
def get_status(temp, threshold=80):
    return "горячо" if temp > threshold else "норма"

print(get_status(85))      # → горячо (threshold=80)
print(get_status(85, 90))  # → норма

# *args — любое количество аргументов
def total(*values):
    return sum(values)

print(total(100, 200, 300))  # → 600

# **kwargs — именованные аргументы
def show_info(**data):
    for key, val in data.items():
        print(f"{key}: {val}")

show_info(ram=16, cpu="i7")

# Возврат нескольких значений (кортеж)
def analyze_ram(total_gb, used_gb):
    free = total_gb - used_gb
    pct = used_gb / total_gb * 100
    return free, pct

free, pct = analyze_ram(16, 11)
print(f"Свободно: {free} GB ({pct:.1f}%)")
```

## Practice

1. Напиши функцию `clamp(value, min_val, max_val)` — возвращает значение в пределах
2. Напиши функцию `stats(numbers)` — возвращает словарь с min, max, avg

## Answers

```python
# 1
def clamp(value, min_val, max_val):
    return max(min_val, min(value, max_val))

print(clamp(150, 0, 100))  # → 100

# 2
def stats(numbers):
    return {
        "min": min(numbers),
        "max": max(numbers),
        "avg": sum(numbers) / len(numbers)
    }

print(stats([10, 20, 30]))  # → {'min': 10, 'max': 30, 'avg': 20.0}
```

## Tips

- Дефолтные аргументы вычисляются один раз! Не используй `def f(x=[])`
- Правильно: `def f(x=None): x = x or []`
- Type hints `def f(x: int) -> str:` не проверяются, но помогают читать код
