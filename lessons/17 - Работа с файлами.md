## Главная идея

Файлы — это способ сохранять данные между запусками программы. Без файлов каждый раз при закрытии программы все данные теряются. PC Booster будет сохранять логи, настройки, профили — всё через файлы.

────────────────────
## Открытие файла — open()

```python
open(путь, режим, encoding="utf-8")
```

Режим • Что делает
`"r"` • Читать (файл должен существовать)
`"w"` • Писать (создаёт новый или перезаписывает)
`"a"` • Дописывать в конец
`"x"` • Создать новый (ошибка если уже есть)
`"rb"` • Читать как байты (картинки, бинарники)

────────────────────
## Чтение файлов

```python
# Способ 1 — with (рекомендуется, файл закрывается автоматически)
with open("report.txt", "r", encoding="utf-8") as f:
    content = f.read()         # всё содержимое как строка
    print(content)

# Способ 2 — построчно
with open("log.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())    # strip() убирает \n в конце

# Способ 3 — список строк
with open("log.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()      # список строк
```

## JSON — сохранение словарей и списков

```python
import json

# Словарь → файл
settings = {
    "profile":  "game",
    "max_temp": 85,
    "auto_boost": True,
    "startup_disable": ["Spotify", "OneDrive"],
}

with open("settings.json", "w", encoding="utf-8") as f:
    json.dump(settings, f, indent=4, ensure_ascii=False)

# Файл → словарь
with open("settings.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)

print(loaded["profile"])     # → game
print(loaded["max_temp"])    # → 85
```

Файл `settings.json` будет выглядеть:
```json
{
    "profile": "game",
    "max_temp": 85,
    "auto_boost": true,
    "startup_disable": ["Spotify", "OneDrive"]
}
```

## Обработка ошибок при работе с файлами

```python
try:
    with open("config.json", "r", encoding="utf-8") as f:
        config = json.load(f)
except FileNotFoundError:
    print("Файл не найден — создаём стандартный конфиг")
    config = {"profile": "normal", "max_temp": 85}
except json.JSONDecodeError:
    print("Файл повреждён — используем дефолт")
    config = {"profile": "normal", "max_temp": 85}
```

## Практика — PC Booster

```python
import json
from datetime import datetime

LOG_FILE      = "boost_log.txt"
SETTINGS_FILE = "settings.json"

def save_settings(settings: dict):
    with open(SETTINGS_FILE, "w", encoding="utf-8") as f:
        json.dump(settings, f, indent=4, ensure_ascii=False)
    print("Настройки сохранены")

def load_settings() -> dict:
    defaults = {"profile": "normal", "max_temp": 85, "auto_boost": False}
    try:
        with open(SETTINGS_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    except (FileNotFoundError, json.JSONDecodeError):
        return defaults

def write_log(action: str):
    timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    with open(LOG_FILE, "a", encoding="utf-8") as f:
        f.write(f"[{timestamp}] {action}\n")

def read_log(last_n=5):
    try:
        with open(LOG_FILE, "r", encoding="utf-8") as f:
            lines = f.readlines()
        return [l.strip() for l in lines[-last_n:]]
    except FileNotFoundError:
        return []

# Использование
save_settings({"profile": "game", "max_temp": 90, "auto_boost": True})
write_log("Профиль изменён на: game")
write_log("Буст активирован")
write_log("Очищена папка temp: 1.2 GB")

settings = load_settings()
print(f"Текущий профиль: {settings['profile']}")

print("\nПоследние записи лога:")
for line in read_log():
    print(f"  {line}")
```

## Задание

PC Booster хранит настройки в JSON-файле. Напиши функции для сохранения и загрузки профилей.

**Дано:**
```python
profiles = {
    "game": {
        "name": "Игровой",
        "cpu_priority": "high",
        "clear_ram": True
    },
    "work": {
        "name": "Рабочий",
        "cpu_priority": "normal",
        "clear_ram": False
    }
}
```

**Нужно:**
1. Сохранить `profiles` в файл `profiles.json` с форматированием (indent=4)
2. Загрузить обратно и вывести
3. Обработать случай если файл не существует

**Ожидаемый вывод:**
```
Сохранено в profiles.json
Загружено из profiles.json:
  game  → Игровой  (high)
  work  → Рабочий  (normal)
```

💡 Подсказка 1
> `json.dump(profiles, f, indent=4, ensure_ascii=False)` — запись с форматированием и русскими символами.

💡 Подсказка 2
> Оберни чтение в `try/except FileNotFoundError` — если файла нет, используй пустой словарь.

💡 Подсказка 3
> `json.load(f)` читает JSON из файла. `json.loads(string)` — из строки. С `s` на конце — строка.

## Решение

```python
import json

PROFILES_FILE = "profiles.json"

def save_profiles(profiles: dict):
    with open(PROFILES_FILE, "w", encoding="utf-8") as f:
        json.dump(profiles, f, indent=4, ensure_ascii=False)
    print("Сохранено в profiles.json")

def load_profiles() -> dict:
    try:
        with open(PROFILES_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    except FileNotFoundError:
        print("Файл не найден — возвращаю пустой словарь")
        return {}

# Сохраняем
profiles = {
    "game": {"name": "Игровой",  "cpu_priority": "high",   "clear_ram": True},
    "work": {"name": "Рабочий",  "cpu_priority": "normal", "clear_ram": False},
}
save_profiles(profiles)

# Загружаем
loaded = load_profiles()
print(f"Загружено из {PROFILES_FILE}:")
for key, val in loaded.items():
    print(f"  {key:<5} → {val['name']:<10} ({val['cpu_priority']})")
```

**Разбор:**
1. `json.dump(data, file, indent=4)` → записывает словарь в файл с отступами
2. `ensure_ascii=False` → русские символы сохраняются как есть, а не как `\u041f`
3. `try/except FileNotFoundError` → если файла нет, возвращаем пустой словарь вместо краша
4. `json.load(file)` → читает JSON и превращает обратно в Python-словарь
5. `for key, val in loaded.items()` → итерация по словарю: ключ и значение

## Заметки на полях

⚠️ Всегда указывай encoding="utf-8"
> Без этого на Windows файлы могут читаться с ошибками из-за кодировки cp1251.
> `open("file.txt", "r", encoding="utf-8")` — всегда.

💡 Проверить существование файла
> ```python
> import os
> if os.path.exists("settings.json"):
>     # файл есть
> ```
> Или просто используй `try/except FileNotFoundError` — питоновский стиль.

📝 `json.dumps()` vs `json.dump()`
> `json.dumps(data)` → возвращает строку JSON.
> `json.dump(data, file)` → пишет в файл.
> Запомни: с `s` на конце — строка (string), без `s` — файл.
