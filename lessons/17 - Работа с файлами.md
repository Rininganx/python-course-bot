# Работа с файлами

> Файлы сохраняют данные между запусками программы.

## Theory
Файлы позволяют хранить настройки, логи и профили. Python работает с файлами через функцию `open()`. Важно всегда указывать кодировку `encoding="utf-8"` для совместимости с Windows.

## Code
```python
# Открытие и чтение файла
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()  # всё содержимое

# Чтение построчно
with open("log.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

# Запись в файл (перезапись)
with open("report.txt", "w", encoding="utf-8") as f:
    f.write("Отчёт\n")

# Дописывание в конец
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("Новая запись\n")

# JSON — сохранение словарей
import json

settings = {"profile": "game", "max_temp": 85}
with open("settings.json", "w", encoding="utf-8") as f:
    json.dump(settings, f, indent=4, ensure_ascii=False)

# Чтение JSON
with open("settings.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
print(loaded["profile"])
```

## Practice
1. Сохрани словарь с настройками в JSON-файл.
2. Прочитай JSON-файл и вывести его содержимое.
3. Обработай ошибку, если файл не существует.

## Answers
```python
import json

# 1. Сохранение
settings = {"theme": "dark", "volume": 80}
with open("config.json", "w", encoding="utf-8") as f:
    json.dump(settings, f, indent=4)

# 2. Чтение
try:
    with open("config.json", "r", encoding="utf-8") as f:
        data = json.load(f)
    print(data)
except FileNotFoundError:
    print("Файл не найден")
```

## Tips
- Всегда используй `encoding="utf-8"` для избежания ошибок кодировки на Windows.
- `json.dump()` пишет в файл, `json.dumps()` возвращает строку.
- Используй `with open(...)` для автоматического закрытия файла.
