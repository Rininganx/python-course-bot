# Модуль os

> Модуль `os` даёт доступ к операционной системе: пути, папки, переменные среды.

## Theory
Модуль `os` используется для работы с файловой системой, получения информации о системе и управления путями. Он позволяет создавать, удалять папки, читать переменные среды и обходить дерево каталогов.

## Code
```python
import os

# Текущая папка и информация о системе
print(os.getcwd())  # текущая папка
print(os.name)      # 'nt' для Windows

# Работа с путями
path = os.path.join("C:\\Users", "User", "Temp")
print(os.path.exists(path))  # проверка существования
print(os.path.isfile(path))  # является ли файлом
print(os.path.isdir(path))   # является ли папкой

# Создание папок
os.makedirs("logs/2026", exist_ok=True)

# Список содержимого папки
for item in os.listdir("C:\\"):
    print(item)

# Обход дерева папок
for root, dirs, files in os.walk("C:\\Users\\User\\Documents"):
    for file in files:
        print(os.path.join(root, file))

# Переменные среды
username = os.environ.get("USERNAME", "Unknown")
temp_dir = os.environ.get("TEMP", "C:\\Temp")
```

## Practice
1. Напиши функцию для получения списка файлов в папке.
2. Создай папку для логов с текущей датой.
3. Прочитай переменную среды `APPDATA`.

## Answers
```python
import os
from datetime import datetime

# 1. Список файлов
def list_files(folder):
    if os.path.exists(folder):
        return os.listdir(folder)
    return []

# 2. Создание папки с датой
date_folder = os.path.join("logs", datetime.now().strftime("%Y-%m-%d"))
os.makedirs(date_folder, exist_ok=True)

# 3. Чтение переменной среды
appdata = os.environ.get("APPDATA", "Не найдено")
print(f"APPDATA: {appdata}")
```

## Tips
- Всегда используй `os.path.join()` для сборки путей — это работает кроссплатформенно.
- Обрабатывай `PermissionError` при обходе системных папок.
- `os.walk()` полезен для рекурсивного обхода всех файлов в дереве папок.
