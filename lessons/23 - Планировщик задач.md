# Планировщик задач

> Планировщик запускает скрипты по расписанию.

## Theory
Планировщик задач Windows позволяет запускать программы при входе в систему, по расписанию или при определённых событиях. Через команду `schtasks` можно создавать, удалять и запускать задачи.

## Code
```python
import subprocess

# Обёртка для schtasks
def run_schtasks(args):
    result = subprocess.run(
        ["schtasks"] + args,
        capture_output=True, text=True, encoding="cp866"
    )
    return result.returncode, result.stdout

# Создание задачи (автозапуск при входе)
code, _ = run_schtasks([
    "/Create", "/TN", "MyTask",
    "/TR", "python script.py",
    "/SC", "ONLOGON", "/F"
])
print("Задача создана" if code == 0 else "Ошибка")

# Удаление задачи
code, _ = run_schtasks(["/Delete", "/TN", "MyTask", "/F"])

# Запуск задачи сейчас
code, _ = run_schtasks(["/Run", "/TN", "MyTask"])

# Просмотр задач
code, output = run_schtasks(["/Query", "/FO", "LIST"])
print(output)
```

## Practice
1. Создай задачу для запуска скрипта при входе в систему.
2. Удали созданную задачу.
3. Проверь, существует ли задача.

## Answers
```python
# 1. Создание задачи
code, _ = run_schtasks([
    "/Create", "/TN", "AutoStart",
    "/TR", "python main.py",
    "/SC", "ONLOGON", "/F"
])
print("Создано" if code == 0 else "Ошибка")

# 2. Удаление задачи
code, _ = run_schtasks(["/Delete", "/TN", "AutoStart", "/F"])
print("Удалено" if code == 0 else "Не найдено")

# 3. Проверка существования
code, _ = run_schtasks(["/Query", "/TN", "AutoStart"])
exists = code == 0
print(f"Задача существует: {exists}")
```

## Tips
- Используй `/F` для перезаписи существующих задач без подтверждения.
- Для пользовательских задач используй `/RL LIMITED` — не требует прав админа.
- Просматривай задачи через `taskschd.msc` (Win+R).
