# Классы и ООП

> Классы — шаблоны для создания объектов с данными и поведением.

## Theory
Объектно-ориентированное программирование (ООП) позволяет моделировать реальные объекты. Класс определяет атрибуты (данные) и методы (поведение). Наследование позволяет создавать новые классы на основе существующих.

## Code
```python
# Базовый класс
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Привет, я {self.name}"

# Создание объекта
person = Person("Анна", 25)
print(person.greet())

# Наследование
class Student(Person):
    def __init__(self, name, age, grade):
        super().__init__(name, age)
        self.grade = grade

    def study(self):
        return f"{self.name} учится в {self.grade} классе"

student = Student("Иван", 16, "10")
print(student.study())

# Магические методы
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)

    def __str__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(1, 2)
v2 = Vector(3, 4)
print(v1 + v2)  # Vector(4, 6)
```

## Practice
1. Создай класс `Dog` с методом `bark()`.
2. Создай класс `Calculator` с методами `add()` и `multiply()`.
3. Используй наследование для класса `Cat` от `Animal`.

## Answers
```python
# 1. Класс Dog
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return f"{self.name}: Гав!"

dog = Dog("Шарик")
print(dog.bark())

# 2. Класс Calculator
class Calculator:
    def add(self, a, b):
        return a + b

    def multiply(self, a, b):
        return a * b

calc = Calculator()
print(calc.add(2, 3))
print(calc.multiply(4, 5))

# 3. Наследование
class Animal:
    def speak(self):
        return "..."

class Cat(Animal):
    def speak(self):
        return "Мяу!"

cat = Cat()
print(cat.speak())
```

## Tips
- `__init__` — конструктор, вызывается при создании объекта.
- `super()` используется для вызова методов родительского класса.
- Магические методы (`__add__`, `__str__`) позволяют использовать объекты с встроенными операциями.
