## Главная идея

Класс — это **шаблон** для создания объектов. Объект combines данные (атрибуты) и поведение (методы). ООП позволяет моделировать реальные вещи: кнопку, файл, пользователя, процесс.

────────────────────
## Базовый синтаксис

```python
class Person:
    def __init__(self, name, age):
        self.name = name      # атрибут
        self.age = age        # атрибут

    def greet(self):          # метод
        return f"Привет, я {self.name}, мне {self.age} лет"

# Создаём объект (экземпляр класса)
person = Person("Анна", 25)
print(person.greet())   # Привет, я Анна, мне 25 лет
print(person.name)      # Анна
```

## Методы

```python
class Calculator:
    def __init__(self):
        self.history = []

    def add(self, a, b):
        result = a + b
        self.history.append(f"{a} + {b} = {result}")
        return result

    def multiply(self, a, b):
        result = a * b
        self.history.append(f"{a} * {b} = {result}")
        return result

    def show_history(self):
        for entry in self.history:
            print(entry)

calc = Calculator()
calc.add(3, 5)
calc.multiply(4, 7)
calc.show_history()
```

```
3 + 5 = 8
4 * 7 = 28
```

## super() — вызов родительского класса

```python
class Vehicle:
    def __init__(self, brand, speed):
        self.brand = brand
        self.speed = speed

    def info(self):
        return f"{self.brand}, {speed} км/ч"

class Car(Vehicle):
    def __init__(self, brand, speed, doors):
        super().__init__(brand, speed)  # вызываем __init__ родителя
        self.doors = doors

    def info(self):
        return f"{self.brand}, {self.speed} км/ч, {self.doors} дверей"

car = Car("Toyota", 180, 4)
print(car.info())   # Toyota, 180 км/ч, 4 дверей
```

## Инкапсуляция (скрытие данных)

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # __ = приватный атрибут

    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount

    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return True
        return False

    def get_balance(self):
        return self.__balance

acc = BankAccount("Иван", 1000)
acc.deposit(500)
print(acc.get_balance())   # 1500
# acc.__balance  →  AttributeError (нельзя напрямую)
```

## Магические методы

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):          # v1 + v2
        return Vector(self.x + other.x, self.y + other.y)

    def __str__(self):                 # print(v)
        return f"Vector({self.x}, {self.y})"

    def __len__(self):                 # len(v)
        return int((self.x**2 + self.y**2)**0.5)

v1 = Vector(3, 4)
v2 = Vector(1, 2)
v3 = v1 + v2
print(v3)        # Vector(4, 6)
print(len(v1))   # 5
```

## Практический пример: Системный монитор

```python
import psutil

class SystemInfo:
    def __init__(self):
        self.cpu_count = psutil.cpu_count()

    def cpu_percent(self):
        return psutil.cpu_percent(interval=1)

    def ram_percent(self):
        return psutil.virtual_memory().percent

    def disk_percent(self, path="C:\\"):
        return psutil.disk_usage(path).percent

    def summary(self):
        return (
            f"CPU: {self.cpu_percent()}% | "
            f"RAM: {self.ram_percent()}% | "
            f"Диск: {self.disk_percent()}%"
        )

monitor = SystemInfo()
print(monitor.summary())
# CPU: 15.2% | RAM: 62.1% | Диск: 45.8%
```

## Задание

Создай класс `TaskManager`:
1. `__init__` — инициализирует пустой список задач
2. `add_task(title)` — добавляет задачу (словарь с `title` и `done=False`)
3. `complete_task(index)` — помечает задачу как выполненную
4. `show_tasks()` — выводит все задачи с номерами и статусами
5. `pending_count` — возвращает количество невыполненных задач

- Задачи храни как список словарей: `[{"title": "...", "done": False}]`
- `complete_task` принимает индекс (номер) задачи
- Используй цикл для `show_tasks`

────────────────────
## Решение

```python
class TaskManager:
    def __init__(self):
        self.tasks = []

    def add_task(self, title):
        self.tasks.append({"title": title, "done": False})

    def complete_task(self, index):
        if 0 <= index < len(self.tasks):
            self.tasks[index]["done"] = True

    def show_tasks(self):
        for i, task in enumerate(self.tasks):
            status = "✅" if task["done"] else "❌"
            print(f"{i+1}. {status} {task['title']}")

    @property
    def pending_count(self):
        return sum(1 for t in self.tasks if not t["done"])

# Используем
tm = TaskManager()
tm.add_task("Установить Python")
tm.add_task("Написать Hello World")
tm.add_task("Изучить функции")

tm.complete_task(0)
tm.show_tasks()

print(f"\nОсталось задач: {tm.pending_count}")
```

```
1. ✅ Установить Python
2. ❌ Написать Hello World
3. ❌ Изучить функции

Осталось задач: 2
```

**Разбор:**
- Класс хранит **данные** (список задач) и **логику** (методы работы с ними)
- `@property` — позволяет обращаться как `tm.pending_count` без скобок
- ООП делает код организованным и переиспользуемым
