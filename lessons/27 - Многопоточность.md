# Многопоточность

> Фоновые задачи без заморозки интерфейса

## Theory

Долгие операции в главном потоке замораживают GUI. Решение — `threading.Thread` в фоновом режиме. tkinter не потокобезопасен: обновляй UI только через `root.after(0, callback)`. `daemon=True` — поток завершится с программой.

## Code

```python
import threading
import time

# Проблема: UI зависает
def optimize():
    time.sleep(5)  # заморозит интерфейс

# Решение: отдельный поток
def optimize():
    time.sleep(5)  # работает в фоне

def start_optimize():
    t = threading.Thread(target=optimize, daemon=True)
    t.start()

# Обновление UI из потока — НЕЛЬЗЯ менять виджеты напрямую
def update_ui(text):
    status_lbl.configure(text=text)  # безопасно через after()

def scan_task():
```

## Practice

1. Создай кнопку "Оптимизировать" с прогрессбаром
2. Фоновый поток делает 5 шагов по 1 секунде, обновляя прогрессбар

## Answers

```python
import customtkinter as ctk
import threading
import time

ctk.set_appearance_mode("dark")
app = ctk.CTk()
app.geometry("350x200")

STEPS = 5
status_var = ctk.StringVar(value="Готово")

def optimize_thread():
    for step in range(1, STEPS + 1):
        app.after(0, lambda s=step: status_var.set(f"Шаг {s}/{STEPS}"))
        app.after(0, lambda s=step: progress.set(s / STEPS))
        time.sleep(1)
    app.after(0, lambda: status_var.set("Готово!"))
    app.after(0, lambda: btn.configure(state="normal"))

def start_optimize():
    btn.configure(state="disabled", text="Работаю...")
    progress.set(0)
    threading.Thread(target=optimize_thread, daemon=True).start()

btn = ctk.CTkButton(app, text="Оптимизировать", command=start_optimize, width=250)
btn.pack(pady=(20, 10))

progress = ctk.CTkProgressBar(app, width=250)
progress.pack(pady=5)

ctk.CTkLabel(app, textvariable=status_var).pack(pady=5)
app.mainloop()
```

## Tips

- Всегда `daemon=True` для фоновых потоков — иначе программа не закроется
- Никогда не меняй виджеты из фонового потока — используй `root.after(0, callback)`
- `lambda s=step:` — захват текущего значения в замыкании
