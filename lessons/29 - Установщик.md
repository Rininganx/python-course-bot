# Установщик

> Создание инсталлятора для .exe приложения

## Theory

Установщик делает из .exe нормальное приложение: ярлыки, запись в реестр, удаление через Панель управления. Inno Setup — проще, NSIS — гибче. Начни с Inno Setup.

## Code

```pascal
; PCBooster.iss — скрипт Inno Setup

[Setup]
AppName=PC Booster
AppVersion=1.0.0
AppPublisher=YourName
DefaultDirName={autopf}\PCBooster
DefaultGroupName=PC Booster
OutputBaseFilename=PCBooster_Setup_1.0.0
OutputDir=release
Compression=lzma2
SolidCompression=yes
PrivilegesRequired=admin

[Files]
Source: "dist\PCBooster.exe"; DestDir: "{app}"; Flags: ignoreversion
Source: "assets\*"; DestDir: "{app}\assets"; Flags: recursesubdirs

[Icons]
Name: "{group}\PC Booster"; Filename: "{app}\PCBooster.exe"
Name: "{commondesktop}\PC Booster"; Filename: "{app}\PCBooster.exe"

[Run]
Filename: "{app}\PCBooster.exe"; Description: "Запустить"; Flags: postinstall nowait

[UninstallDelete]
Type: filesandordirs; Name: "{app}\assets"
```

```nsis
; NSIS альтернатива
!define APP_NAME "PC Booster"
Name "${APP_NAME}"
OutFile "Setup.exe"
InstallDir "$PROGRAMFILES\PCBooster"
RequestExecutionLevel admin

Section "Установка"
    SetOutPath "$INSTDIR"
    File "dist\PCBooster.exe"
    CreateShortcut "$DESKTOP\PC Booster.lnk" "$INSTDIR\PCBooster.exe"
    WriteUninstaller "$INSTDIR\Uninstall.exe"
SectionEnd
```

## Practice

1. Напиши .iss файл для Inno Setup с ярлыками на рабочий стол и в меню Пуск
2. Настрои деинсталляцию через `[UninstallDelete]`

## Answers

```pascal
; PCBooster.iss
[Setup]
AppName=PC Booster
AppVersion=1.0.0
DefaultDirName={autopf}\PCBooster
OutputBaseFilename=PCBooster_Setup_1.0.0
OutputDir=release
Compression=lzma2
PrivilegesRequired=admin

[Files]
Source: "dist\PCBooster.exe"; DestDir: "{app}"; Flags: ignoreversion
Source: "assets\*"; DestDir: "{app}\assets"; Flags: recursesubdirs

[Icons]
Name: "{group}\PC Booster"; Filename: "{app}\PCBooster.exe"
Name: "{commondesktop}\PC Booster"; Filename: "{app}\PCBooster.exe"

[Run]
Filename: "{app}\PCBooster.exe"; Description: "Запустить"; Flags: postinstall nowait

[UninstallDelete]
Type: filesandordirs; Name: "{app}\assets"
Type: filesandordirs; Name: "{app}\profiles"
```

## Tips

- `{autopf}` = `C:\Program Files`, `{commondesktop}` = рабочий стол
- Открой .iss в Inno Setup → Build → Compile
- Версионируй файл установщика: `PCBooster_Setup_1.0.0.exe`
