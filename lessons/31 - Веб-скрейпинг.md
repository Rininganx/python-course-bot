# Веб-скрейпинг

> Автоматическое извлечение данных с веб-страниц

## Theory

Скрейпинг — извлечение данных с сайтов. `requests` загружает HTML, `BeautifulSoup` парсит его. CSS-селекторы помогают найти элементы. Для динамических страниц (JavaScript) используй Selenium. Один из самых востребованных навыков на фрилансе.

## Practice

1. Спарси заголовки статей с новостного сайта
2. Сохрани результат в CSV с обработкой ошибок

## Answers

```python
import requests
from bs4 import BeautifulSoup
import csv

def parse_news(url):
    try:
        headers = {"User-Agent": "Mozilla/5.0"}
        response = requests.get(url, headers=headers, timeout=10)
        response.raise_for_status()

        soup = BeautifulSoup(response.text, "html.parser")
        articles = []
        for item in soup.select("article, .post, .news-item"):
            title_el = item.select_one("h1, h2, h3")
            if title_el:
                articles.append({"title": title_el.text.strip()})
        return articles
    except requests.RequestException as e:
        print(f"Ошибка: {e}")
        return []

news = parse_news("https://news.ycombinator.com")
if news:
    with open("news.csv", "w", newline="", encoding="utf-8") as f:
        writer = csv.DictWriter(f, fieldnames=["title"])
        writer.writeheader()
        writer.writerows(news)
    print(f"Сохранено {len(news)} статей")
```

## Tips

- Всегда добавляй `User-Agent` — иначе могут заблокировать
- `timeout=10` — предотвращает зависание запроса
- `response.raise_for_status()` — выбрасывает исключение при ошибках HTTP
