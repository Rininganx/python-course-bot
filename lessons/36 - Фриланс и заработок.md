# Фриланс и заработок на Python

> Как найти заказы и начать зарабатывать от 500 до 5000+ евро в месяц

## Theory

Python — один из самых востребованных языков на фрилансе. Боты, парсеры, автоматизация — заказывают каждый день. Начни с低价 для первых отзывов, потом повышай цены. Специализируйся на 1-2 типах услуг.

## Как создать гиг на Fiverr

**Профиль:**
```
Python Developer | Automation Expert | Bot Creator
Telegram & Discord bots, Web scraping, Excel/PDF automation
Fast delivery, clean code, 24/7 support
```

**Гиг:**
- Название: "I will create a custom Telegram bot"
- Категория: Programming & Tech > Chatbots
- 3 пакета: Basic €50, Standard €150, Premium €300
- Сроки: 3, 5, 7 дней

**Портфолио** — 3-5 работ на GitHub.

## Ценообразование

| Услуга | Basic | Standard | Premium |
|--------|-------|----------|---------|
| Telegram бот | €50 | €150 | €300 |
| Парсер | €30 | €100 | €250 |
| Автоматизация | €40 | €120 | €300 |

Как повышать цены:
1. Начни с低价 — набери 5-10 отзывов
2. Повышай на 20-30% каждые 10 заказов
3. Специализируйся — узкая ниша = выше цены

## Получение первого заказа

1. Создай 3-5 гигов
2. Сделай портфолио на GitHub
3. Отправляй proposals каждый день
4. Отвечай в течение 1 часа
5. Первые заказы — с бонусами

**Шаблон предложения:**
```
Hi! I saw your project and I'm interested.
I can deliver: [Feature 1], [Feature 2]
Price: €X, Timeline: X days
I'm available to start immediately.
```

## План заработка на 6 месяцев

| Месяц | Цель | Действия |
|-------|------|----------|
| 1-2 | €200-500/мес | 3-5 гигов, 5-10 заказов,低价 |
| 3-4 | €500-1500/мес | +30% к ценам, специализация |

## Tips

- Начни с Telegram ботов — самый популярный запрос
- Делай больше за меньше в первых заказах — отзывы важнее денег
- Регистрируй ИП и плати налоги — консультация бухгалтера стоит €50-100
